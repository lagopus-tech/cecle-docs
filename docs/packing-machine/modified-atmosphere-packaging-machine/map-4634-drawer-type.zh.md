### 1. 产品概述
* **产品类别**：食品包装机械 / 气调保鲜包装机。
* **应用场景**：广泛适用于生鲜肉类、熟食卤味、腌制品、果蔬、海鲜、菌菇及冷藏即食餐品。适合中小型食品加工厂、超市后厨及餐饮连锁配送中心。
* **核心功能**：集真空抽气、充气（氮气/混合气体）、热封、切膜、光电对位于一体。通过改变包装内气体环境，抑制细菌滋生，显著延长食品保质期。

### 2. 核心优势
1. **优质部件，性能稳定**：配备**信捷（XINJE）触摸屏与PLC**控制系统，确保高强度运行下的精准度与耐用性。
2. **延长保质期**：高真空能力结合精准气体置换技术，有效防止霉变、腐败和氧化，保留食品原色、原味与口感。
3. **紧凑设计，高卫生标准**：采用优质碳钢与不锈钢制造，符合食品安全标准。占地面积小，便于灵活摆放与移动。
4. **精准切膜**：配备自动光电传感器系统，确保膜图案完美对位，切边整齐美观。

### 4. 模具与耗材规格

#### **1. 模具配置与尺寸**
模具总尺寸为 **640 * 440 mm**。标准机型支持最大托盘高度为 **50 mm**（高度可根据需求定制）。

#### **2. 耗材推荐**
为确保最佳封口质量与保鲜效果，封口膜必须与托盘材质严格匹配：
* **膜规格**：本机支持最大膜宽 **440 mm**。推荐标准膜厚为 **80微米**。
* **PP托盘**：建议使用配套的PP封口膜。
* **PET或CPET托盘**：建议升级使用 **EVOH高阻隔封口膜**。EVOH膜价格较高，但其优异的氧气阻隔性能可显著延长产品保质期。

### 7. 工作流程
1. **人工放盒**：操作员将装有产品的托盘放入模具。
2. **自动进料**：机器自动将模具推入工作腔室。
3. **抽真空与充气**：系统自动抽取空气并充入预设的保鲜混合气体。
4. **热封与切膜**：同步完成热封及切除多余膜边。
5. **光电校正**：传感器识别膜标，确保图案完美居中。
6. **自动出料**：包装完成后，模具自动推出。
7. **人工取盒**：操作员取出成品，重复循环。
