## 1. 产品概述
TBP-180 是一款台式气动泡罩包装机，专为胶囊、药片、小五金件、电子元件等小件物品的 PVC 泡罩 + 铝箔密封而设计。采用气缸压合 + 数显时间控制，运行稳定，操作简单。适用于小批量生产、实验室、商店、工厂以及小型外贸订单。

## 2. 核心优势
- 气动压合：压力均匀，密封牢固，省力高效
- 数显时间控制：加热时间精准，密封质量稳定
- 紧凑台式设计：占地面积小，易于放置和移动，可直接在桌面上使用，支持空运或快递，节省运输成本
- 模具可定制：根据客户泡罩样品制作模具（需提供 6–8 个样品），换模快捷，适用于多种产品
- 宽电压兼容：支持 110V/220V，50/60Hz，全球通用
- 需配空压机：气压稳定，效率高

## 4. 操作步骤
1. 将胶囊、药片等产品手动放入 PVC 泡罩中；
2. 将装有产品的 PVC 泡罩放入机器模具中，然后盖上铝箔膜；
3. 按下设备开关，机器将自动下压完成密封操作；
4. 用剪刀修剪泡罩边缘多余的铝箔膜，即可得到完整的密封泡罩包装。
