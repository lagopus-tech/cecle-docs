## 4. Proceso de Operación
1. Colocar manualmente cápsulas, comprimidos y otros productos en blísteres de PVC;
2. Colocar el blíster de PVC cargado en el molde de la máquina, luego añadir la lámina de aluminio;
3. Encender el interruptor del dispositivo y presionar la palanca manual para completar el sellado;
4. Recortar el exceso de lámina de aluminio alrededor del blíster con un cortador o tijeras para obtener el envase terminado.
