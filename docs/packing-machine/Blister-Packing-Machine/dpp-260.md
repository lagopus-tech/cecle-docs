The DPP-260 automatic alu-plastic blister packing machine is specially designed for continuous forming, filling, heat sealing, batch number printing and punching of PVC/PET blister + aluminum foil for capsules, tablets, pills, small hardware, electronic components, food and other items. Adopting PLC touch screen control and servo drive system, it features stable operation and high automation, suitable for pharmaceutical factories, health care products factories, food factories, standardized production lines and large-scale continuous production.

## 4. Operation Steps
1. Install PVC film roll and aluminum foil roll, adjust the film feeding position and tension;
2. Set forming temperature, heat sealing temperature, running speed and other parameters on the touch screen;
3. Start the machine for preheating and test run without materials, check if forming, sealing and punching are normal;
4. Put in materials, start automatic operation, the machine will automatically complete blister forming, filling, sealing, coding and punching;
5. Automatically collect finished products and waste materials to realize continuous automatic production;
6. After production, stop the machine, cut off power and air supply, clean the equipment and molds.
