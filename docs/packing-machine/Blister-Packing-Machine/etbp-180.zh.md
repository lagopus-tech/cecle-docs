## 4. 操作步骤
1. 将胶囊、药片等产品手动放入PVC泡罩中。
2. 将装有产品的PVC泡罩放入机器模具中。
3. 按下按钮，机器自动下压封合。
4. 机器自动收放铝箔。
5. 用剪刀裁剪泡罩并修边。
6. 得到成品封合产品。
