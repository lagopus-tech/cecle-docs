## 4. Langkah Operasi
1. Masukkan kapsul, tablet, dan produk lainnya secara manual ke dalam PVC blister.
2. Tempatkan PVC blister berisi produk ke dalam cetakan mesin.
3. Tekan tombol, mesin akan secara otomatis menekan ke bawah untuk penyegelan.
4. Mesin secara otomatis menggulung/menggulung ulang aluminium foil.
5. Potong blister dan rapikan tepinya dengan gunting.
6. Dapatkan produk tersegel yang sudah jadi. |
