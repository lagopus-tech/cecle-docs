## 4. Proses Operasi
1. Masukkan kapsul, tablet, dan produk lainnya secara manual ke dalam blister PVC;
2. Tempatkan blister PVC yang sudah diisi ke dalam cetakan mesin, lalu tambahkan film aluminium foil;
3. Nyalakan sakelar perangkat dan tekan tuas manual ke bawah untuk menyelesaikan penyegelan;
4. Potong kelebihan aluminium foil di sekitar blister dengan pemotong atau gunting untuk mendapatkan kemasan jadi.
