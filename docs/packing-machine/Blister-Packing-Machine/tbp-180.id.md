## 4. Langkah Pengoperasian
1. Masukkan kapsul, tablet, dan produk lainnya secara manual ke dalam blister PVC;
2. Tempatkan blister PVC berisi produk ke dalam cetakan mesin, lalu tambahkan film aluminium foil;
3. Tekan sakelar peralatan, mesin akan secara otomatis menekan ke bawah untuk menyelesaikan operasi penyegelan;
4. Potong kelebihan film aluminium foil di tepi blister dengan gunting untuk mendapatkan kemasan blister tersegel yang lengkap.
