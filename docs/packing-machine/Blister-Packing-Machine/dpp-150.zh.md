## 4. 操作步骤
1. 安装 PVC 薄膜卷和铝箔卷，调整薄膜送料位置和张力；
2. 在触摸屏上设置成型温度、热封温度、运行速度等参数；
3. 启动机器进行预热和空载试运行，检查成型、封合和冲裁是否正常；
4. 放入物料，启动自动运行，机器将自动完成泡罩成型、填充、封合、打码和冲裁；
5. 自动收集成品和废料，实现连续自动化生产；
6. 生产结束后，停机，切断电源和气源，清洁设备和模具。
