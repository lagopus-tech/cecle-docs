## 4. Etapas de Operação
1. Coloque manualmente cápsulas, comprimidos e outros produtos no blister de PVC.
2. Posicione o blister de PVC com os produtos no molde da máquina.
3. Pressione o botão, a máquina pressionará automaticamente para selagem.
4. A máquina rebobina/desbobina automaticamente a folha de alumínio.
5. Corte o blister e apare as bordas com tesoura.
6. Obtenha o produto selado finalizado. |
