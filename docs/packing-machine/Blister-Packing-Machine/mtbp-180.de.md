## 4. Betriebsablauf
1. Kapseln, Tabletten und andere Produkte manuell in PVC-Blister einlegen;
2. Den bestückten PVC-Blister in das Maschinenformular einlegen, dann Aluminiumfolie hinzufügen;
3. Geräteschalter einschalten und den manuellen Hebel herunterdrücken, um die Versiegelung abzuschließen;
4. Überschüssige Aluminiumfolie um den Blister mit einem Cutter oder einer Schere abschneiden, um eine fertige Verpackung zu erhalten.
