## 1. 产品概述
DP-180 是一款立式气动铝塑泡罩包装机，专为胶囊、药片及类似产品的铝箔密封而设计。
该机采用 PLC + 触摸屏控制，具备自动送铝箔、光电对版及气动热压升降功能。模具可根据客户泡罩样品定制。机器运行稳定、操作简便，适用于中小批量生产。

---

## 2. 核心优势
1. 气动控制热板升降，动作稳定可靠
2. 自动送铝箔并配备光电对版（机器装有光电传感器，用于检测铝箔上的色标。铝箔采用两段式送料：先快速前进，接近传感器时减速定位色标。检测到色标后电机停止，确保 PVC 泡罩上图案精准对齐，避免偏移，特别适用于印刷铝箔产品，封合美观整齐）
3. PLC + 触摸屏 + 数字时间继电器，实现精确的温度和时间控制
4. 模具更换快捷，维护方便，适用于多种产品
5. 可根据客户泡罩尺寸和形状定制模具
6. 可定制集成封合 + 打码（批号/日期）功能
7. 结构紧凑，占地面积小，操作简单，适合小型工厂

## 4. 操作流程
1. 将胶囊、药片等产品手动放入 PVC 泡罩中
2. 将装好产品的 PVC 泡罩放入机器模具
3. 踩下脚踏板启动热封
4. 机器自动收卷/送铝箔并完成光电对版（光电传感器检测色标，采用两段式速度控制，确保图案精准对齐，避免偏移）
5. 用剪刀或裁切机修边
6. 获得成品密封泡罩包装
