## 1. 产品概述
CP-180 是一款小型气动铝箔-PVC 泡罩包装机，专为胶囊、药片等小型药品的铝塑泡罩包装而设计。也适用于小型五金件、食品等物品的泡罩包装。该机结构紧凑、操作简便，集气动控制、自动送箔和数字温控于一体，适用于小批量生产和实验室使用。可根据样品定制模具，满足差异化包装需求。

## 4. 操作流程
1. 将胶囊、药片等产品手动放入PVC泡罩中。
2. 将装有产品的PVC泡罩放入机器模具中。
3. 踩下脚踏板，机器下压进行热封。
4. 机器自动收卷/送进铝箔。
5. 用剪刀或裁切机裁断泡罩并修边。
6. 得到成品密封产品。

### 5.2 常见问题与解决方案
#### 1. 泡罩成型不良（变形、穿孔、不完整）
- 原因：加热温度不足/过高、气压不稳定、模具不平行、冷却不足。
- 解决方案：校准温度（140-160℃），稳定气压0.6-0.8MPa，调整模具平行度，清洁排气孔，优化冷却水流量。

#### 2. 热封不牢（泄漏、气泡、易撕开）
- 原因：加热时间短、温度低、压力不足、铝箔胶层损坏、加热板有污垢。
- 解决方案：延长加热时间至4-5秒，温度提高5-10℃，调整气动压力，更换优质铝箔，清洁加热板表面污垢。

#### 3. 铝箔起皱与跑偏
- 原因：送箔张力不均、导辊不平行、压杆高度不平衡、铝箔质量差。
- 解决方案：调整送箔张力旋钮，校准导辊平行度，微调压杆高度，更换无褶皱、厚度均匀的铝箔。

#### 4. 生产速度慢、效率低
- 原因：参数设置不合理、手动送料慢、模具适应性差。
- 解决方案：优化加热/封口参数，培训员工快速送料，定制兼容性更高的模具（最大180×140mm / 7.09×5.51英寸）。

#### 5. 机器无法启动/加热板不加热
- 原因：电源连接不良、温控器故障、加热管损坏、未连接气源。
- 解决方案：检查电源线路和插头，校准或更换温控器，维修加热管，确保空压机正常供气且无泄漏。
