## 4. Processo Operativo
1. Inserire manualmente capsule, compresse e altri prodotti nei blister in PVC;
2. Posizionare il blister in PVC caricato nello stampo della macchina, quindi aggiungere il foglio di alluminio;
3. Accendere l'interruttore del dispositivo e premere la leva manuale per completare la sigillatura;
4. Tagliare il foglio di alluminio in eccesso attorno al blister con un taglierino o delle forbici per ottenere la confezione finita.
