## 1. 产品概述
BP-180 是一款半自动平板式铝箔 PVC 泡罩包装机，专为胶囊、药片等小型固体产品的铝箔-PVC 泡罩密封包装而设计。该机型为基础经济型，采用气动驱动和数字温控技术，结构紧凑、操作简单、维护成本低，适用于中小批量生产，广泛应用于制药、保健品、电子元件等行业。

## 2. 核心优势
- **专业设计，适应性强**：专为胶囊铝箔包装定制，最大模具尺寸为 180mm×140mm（7.09"×5.51"），兼容相应尺寸的 PVC 泡罩和铝箔材料。
- **气动驱动，稳定高效**：气动控制加热板升降，压力均匀（0.4T），热封牢固。配备数字时间继电器，精确控制加热时间（3-5s），确保密封性能优异。
- **快速换模，灵活定制**：可快速更换模具以适应不同产品生产。模具根据泡罩尺寸/形状定制，支持批量定制带批号和日期的压印模具。
- **操作简便，门槛低**：通过脚踏开关控制。操作人员只需手动放置铝箔片，无需专业技能，易于掌握。
- **经久耐用，售后无忧**：标准化工厂生产，核心部件质量可控。提供 1 年保修（不含易损件）。
- **紧凑节能，成本低廉**：机身轻巧，占用空间小，功率 1000W。无需复杂安装，连接外置空压机即可使用。

## 4. 操作流程
1. 将胶囊、药片等产品手动放入 PVC 泡罩中。
2. 将装有产品的 PVC 泡罩放入机器模具，然后放置一片铝箔。
3. 踩下脚踏开关，机器下压进行热封。
4. 用剪刀或裁切机裁切泡罩并修边。
5. 获得成品密封包装。

**问：BP-180 和 CP-210 有什么区别？**
答：BP-180 为基础机型，无自动送铝箔功能，需手动逐片放置铝箔；CP-210 为升级版，支持自动送铝箔，效率更高，价格也更高。

**问：机器能否打印批号和生产日期？**
答：可以，模具可定制压印功能，实现热封与批号/日期同步打印。

**问：设备售后维护方便吗？**
答：提供一站式售后服务，1 年保修（不含易损件），保修期后终身维护。可通过邮件反馈问题，支持工程师海外现场服务。
