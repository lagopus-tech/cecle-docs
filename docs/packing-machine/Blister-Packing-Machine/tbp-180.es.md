## 1. Descripción General del Producto
La TBP-180 es una máquina de envasado en blíster neumática de sobremesa, especialmente diseñada para el sellado de blíster de PVC + lámina de aluminio de artículos pequeños como cápsulas, comprimidos, ferretería pequeña y componentes electrónicos. Utiliza prensado por cilindro + control de tiempo con pantalla digital, ofreciendo un funcionamiento estable y una operación sencilla. Es adecuada para producción en lotes pequeños, laboratorios, tiendas, fábricas y pedidos pequeños de comercio exterior.

## 2. Ventajas Clave
- Prensado Neumático: Presión uniforme, sellado firme, ahorro de mano de obra y alta eficiencia
- Control de Tiempo con Pantalla Digital: Tiempo de calentamiento preciso, calidad de sellado estable
- Diseño Compacto de Sobremesa: Huella pequeña, fácil colocación y movimiento, se puede usar directamente sobre la mesa, admite transporte aéreo o mensajería para ahorrar costes de envío
- Moldes Personalizables: Fabricación de moldes según muestras de blíster del cliente (se requieren 6–8 muestras), cambio rápido de molde, adecuado para múltiples productos
- Amplia Compatibilidad de Voltaje: Soporta 110V/220V, 50/60Hz para uso global
- Requiere Compresor de Aire: Presión de aire estable, alta eficiencia

## 4. Pasos de Operación
1. Colocar manualmente cápsulas, comprimidos y otros productos en el blíster de PVC;
2. Colocar el blíster de PVC con los productos en el molde de la máquina, luego añadir la lámina de aluminio;
3. Presionar el interruptor del equipo, la máquina presionará automáticamente hacia abajo para completar la operación de sellado;
4. Recortar el exceso de lámina de aluminio en el borde del blíster con tijeras para obtener un paquete de blíster sellado completo.
