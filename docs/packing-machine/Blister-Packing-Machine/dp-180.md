# DP-180 Vertical Blister Packing Machine

---

## 1. Product Overview
The DP-180 is a vertical pneumatic aluminum-plastic blister packing machine specially designed for aluminum foil sealing of capsules, tablets, and similar products.
It adopts PLC + touch screen control, automatic aluminum foil feeding, photoelectric registration, and pneumatic hot pressing lift. Molds can be customized according to customer blister samples. The machine runs stably, is easy to operate, and is suitable for small and medium batch production.

---

## 2. Core Advantages
1. Pneumatic control for hot platen lifting, with stable and reliable action
2. Automatic aluminum foil feeding with photoelectric registration (the machine is equipped with a photoelectric sensor to detect the color mark on the aluminum foil. The aluminum foil adopts two-speed feeding: it moves fast first, then slows down to locate the color mark when approaching the sensor. The motor stops once the color mark is detected, ensuring accurate pattern alignment on PVC blisters and avoiding offset, making it ideal for printed aluminum foil products with neat and beautiful sealing)
3. PLC + touch screen + digital time relay for precise temperature and time control
4. Quick mold replacement, easy maintenance, suitable for various products
5. Mold customization according to customer blister size and shape
6. Customizable integrated sealing + coding (batch number / date) function
7. Compact structure, small footprint, simple operation, suitable for small factories

## 4. Operation Process
1. Manually put capsules, tablets and other products into PVC blisters
2. Place the loaded PVC blister into the machine mold
3. Step on the foot pedal to start hot sealing
4. The machine automatically rewinds / feeds aluminum foil and completes photoelectric registration (the photoelectric sensor detects color marks with two-speed control to ensure accurate pattern alignment and avoid offset)
5. Trim edges with scissors or cutting machine
6. Get the finished sealed blister package
