## 1. Обзор продукта
DP-180 — это вертикальная пневматическая блистерная упаковочная машина для алюминиево-пластиковой упаковки, специально разработанная для герметизации алюминиевой фольгой капсул, таблеток и аналогичных продуктов.
Она оснащена управлением ПЛК + сенсорный экран, автоматической подачей алюминиевой фольги, фотоэлектрической регистрацией и пневматическим подъемом горячего пресса. Пресс-формы могут быть изготовлены по индивидуальному заказу в соответствии с образцами блистеров заказчика. Машина работает стабильно, проста в эксплуатации и подходит для мелко- и среднесерийного производства.

## 2. Ключевые преимущества
1. Пневматическое управление подъемом горячей плиты, стабильное и надежное действие
2. Автоматическая подача алюминиевой фольги с фотоэлектрической регистрацией (машина оснащена фотоэлектрическим датчиком для обнаружения цветной метки на алюминиевой фольге. Алюминиевая фольга подается на двух скоростях: сначала быстро, затем замедляется для позиционирования цветной метки при приближении к датчику. Двигатель останавливается после обнаружения цветной метки, обеспечивая точное совмещение рисунка на ПВХ-блистерах и исключая смещение, что идеально подходит для продукции с печатной алюминиевой фольгой, обеспечивая аккуратную и красивую герметизацию)
3. ПЛК + сенсорный экран + цифровое реле времени для точного контроля температуры и времени
4. Быстрая замена пресс-форм, простота обслуживания, подходит для различных продуктов
5. Изготовление пресс-форм по индивидуальному заказу в соответствии с размером и формой блистера заказчика
6. Возможность настройки интегрированной функции герметизации + нанесения кода (номер партии / дата)
7. Компактная конструкция, малая занимаемая площадь, простота эксплуатации, подходит для небольших производств

## 4. Процесс работы
1. Вручную поместите капсулы, таблетки и другие продукты в ПВХ-блистеры
2. Поместите заполненный ПВХ-блистер в пресс-форму машины
3. Нажмите на ножную педаль для начала термосваривания
4. Машина автоматически перематывает / подает алюминиевую фольгу и выполняет фотоэлектрическую регистрацию (фотоэлектрический датчик обнаруживает цветные метки с двухскоростным управлением для обеспечения точного совмещения рисунка и исключения смещения)
5. Обрежьте края ножницами или резаком
6. Получите готовую герметичную блистерную упаковку

### Часто задаваемые вопросы
1. **В: Входит ли компрессор в комплект поставки?**
О: Нет, требуется внешний компрессор 1200 Вт × 50 л.

2. **В: Можно ли менять пресс-формы для разных размеров?**
О: Одна пресс-форма для одного размера; доступны новые пресс-формы по индивидуальному заказу.

3. **В: Можно ли печатать номер партии / дату?**
О: Да, пресс-форма по индивидуальному заказу поддерживает интегрированную герметизацию + нанесение кода.

4. **В: Какой срок гарантии?**
О: 1 год гарантии на всю машину (за исключением быстроизнашивающихся деталей), пожизненное обслуживание.

5. **В: Какие услуги предоставляются?**
О: Онлайн-поддержка, видеоподдержка, бесплатные запасные части в течение гарантийного срока.
