## 4. 操作流程
1. 手动将胶囊、片剂等产品放入PVC泡罩中；
2. 将装好产品的PVC泡罩放入机器模具中，然后盖上铝箔膜；
3. 打开设备开关，按下手动杠杆完成封合；
4. 用切刀或剪刀修剪泡罩周围多余的铝箔，即可得到成品包装。
