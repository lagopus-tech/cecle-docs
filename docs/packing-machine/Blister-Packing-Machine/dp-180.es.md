## 1. Descripción General del Producto
La DP-180 es una máquina envasadora vertical neumática de blíster de aluminio-plástico, diseñada especialmente para el sellado con lámina de aluminio de cápsulas, comprimidos y productos similares.
Adopta control PLC + pantalla táctil, alimentación automática de lámina de aluminio, registro fotoeléctrico y elevación neumática de la placa caliente. Los moldes se pueden personalizar según las muestras de blíster del cliente. La máquina funciona de manera estable, es fácil de operar y es adecuada para producción en lotes pequeños y medianos.

---

## 2. Ventajas Principales
1. Control neumático para la elevación de la placa caliente, con acción estable y fiable
2. Alimentación automática de lámina de aluminio con registro fotoeléctrico (la máquina está equipada con un sensor fotoeléctrico para detectar la marca de color en la lámina de aluminio. La lámina de aluminio adopta una alimentación de dos velocidades: primero se mueve rápido, luego reduce la velocidad para localizar la marca de color al acercarse al sensor. El motor se detiene una vez que se detecta la marca de color, asegurando una alineación precisa del patrón en los blísteres de PVC y evitando desviaciones, lo que lo hace ideal para productos de lámina de aluminio impresa con un sellado ordenado y estético)
3. PLC + pantalla táctil + relé de tiempo digital para un control preciso de temperatura y tiempo
4. Cambio rápido de molde, fácil mantenimiento, adecuado para varios productos
5. Personalización de moldes según el tamaño y la forma del blíster del cliente
6. Función integrada de sellado + codificación (número de lote / fecha) personalizable
7. Estructura compacta, huella pequeña, operación simple, adecuado para pequeñas fábricas

## 4. Proceso de Operación
1. Colocar manualmente cápsulas, comprimidos y otros productos en los blísteres de PVC
2. Colocar el blíster de PVC cargado en el molde de la máquina
3. Pisar el pedal para iniciar el sellado por calor
4. La máquina rebobina / alimenta automáticamente la lámina de aluminio y completa el registro fotoeléctrico (el sensor fotoeléctrico detecta las marcas de color con control de dos velocidades para asegurar una alineación precisa del patrón y evitar desviaciones)
5. Recortar los bordes con tijeras o una máquina cortadora
6. Obtener el envase de blíster sellado terminado
