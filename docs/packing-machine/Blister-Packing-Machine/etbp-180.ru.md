## 4. Этапы работы
1. Вручную поместите капсулы, таблетки и другие продукты в ПВХ-блистер.
2. Поместите ПВХ-блистер с продуктами в пресс-форму машины.
3. Нажмите кнопку, машина автоматически опустится для запайки.
4. Машина автоматически перематывает/разматывает алюминиевую фольгу.
5. Разрежьте блистер и обрежьте края ножницами.
6. Получите готовый запаянный продукт. |
