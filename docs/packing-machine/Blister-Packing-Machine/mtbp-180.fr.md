## 4. Processus de Fonctionnement
1. Placer manuellement les gélules, comprimés et autres produits dans les alvéoles PVC ;
2. Placer l'alvéole PVC chargée dans le moule de la machine, puis ajouter le film en aluminium ;
3. Allumer l'interrupteur de l'appareil et abaisser le levier manuel pour effectuer le scellage ;
4. Couper l'excédent de feuille d'aluminium autour de l'alvéole avec un cutter ou des ciseaux pour obtenir un emballage fini.
