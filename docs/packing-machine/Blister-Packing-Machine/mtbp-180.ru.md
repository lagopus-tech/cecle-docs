## 4. Процесс работы
1. Вручную поместите капсулы, таблетки и другие продукты в ПВХ-блистеры;
2. Поместите заполненный ПВХ-блистер в пресс-форму машины, затем добавьте пленку из алюминиевой фольги;
3. Включите переключатель устройства и нажмите ручной рычаг для завершения запайки;
4. Обрежьте излишки алюминиевой фольги вокруг блистера ножом или ножницами, чтобы получить готовую упаковку.
