## I. Обзор продукта

### 1. Категория продукта
* **Наименование оборудования**: Плоский трафаретный принтер
* **Модель продукта**: TD-1525
* **Позиционирование оборудования**: Пневматическое трафаретное печатное оборудование, специально разработанное для небольших плоских изделий.
* **Форм-фактор**: **Настольная конструкция**. Вес нетто станка составляет около 65 кг. Рекомендуется размещать на прочном верстаке с достаточной несущей способностью, соответствующем эргономическим требованиям по высоте, для удобства оператора при совмещении и съеме материала.

### 2. Область применения
* **Отраслевое применение**: Широко используется для печати на различных мелких плоских объектах (например, акриловые таблички, стеклянные линзы, корпуса электронных компонентов, приборные панели и т.д.).
* **Характеристики материала**: Поддерживает подложки толщиной до 120 мм (12 см), обладает чрезвычайно высокой совместимостью.
* **Масштаб производства**: Идеально подходит для мелко- и среднесерйной обработки партий. Обеспечивает стабильную производительность и высокий выход годной продукции.

### 3. Основные функции
* **Прецизионный стол 0,01 мм**: Оснащен ручками точной настройки вперед/назад, влево/вправо, строго гарантируя точность позиционирования по осям X/Y до 0,01 мм.
* **Система подъема на направляющих**: Подъем рамы трафарета осуществляется по направляющим, что обеспечивает высокую точность и плавность хода.
* **Высокоэффективная пневматическая печать**: Приводится в действие сжатым воздухом 4 БАР, работает чрезвычайно быстро, максимальная производительность печати до 1500 шт/ч, что выше, чем у обычных плоскостных моделей.

## IV. Эксплуатация и рабочий процесс

### 1. Необходимые материалы
Перед эксплуатацией данного станка клиентам необходимо самостоятельно подготовить или приобрести у нас следующие расходные материалы и вспомогательное оборудование:
* **Воздушный компрессор**: Обязателен. Рекомендуется оснастить компрессором объемом около 90 л и мощностью около 2,55 кВт для обеспечения пневматического источника питания станка.
* **Трафаретная сетка**: Необходимо изготовить индивидуальную трафаретную сетку в соответствии с рисунком для печати.
* **Краска**: Выберите краску с соответствующими свойствами в зависимости от материала печатного изделия (например, металл, стекло и т.д.).
* **Разбавитель**: Используется для регулировки вязкости краски, обеспечивая плавную печать.
* **Ракель**: Устанавливается на станок для протягивания и прижатия краски.
* **Смывка для трафарета**: Используется для очистки остатков краски с трафарета после печати.
* **Приспособление**: Используется для точной фиксации изделия на столе.

### 2. Стандартный рабочий процесс
1. **Установка трафарета**: Надежно закрепите трафаретную сетку с индивидуальным рисунком в зажимах рамы станка.
2. **Размещение изделия**: Поместите плоское изделие для печати на рабочий стол.
3. **Добавление краски**: Налейте необходимое количество подготовленной краски на трафаретную сетку.
4. **Запуск печати**: Запустите процесс с помощью панели управления или ножной педали. Станок автоматически опустится по направляющим и выполнит печать рисунка.
