## 二、操作准备
本系列所有机器均为**纯机械手动操作**，零能耗，无需空压机或电源。
* **所需耗材**：定制图案的网版、适用于特定材料的油墨、刮刀以及洗网水等清洁耗材。


## 三、销售技巧与常见问题

### 销售话术与建议
* **精准询盘与引导**：当客户发来询盘时，您的第一句话应该是：“您好！请问您要印刷什么产品，需要印刷几种颜色？”
**印刷平面/单色** ➡️ 重点推荐 **MS / (A)MS系列**。
**印刷服装/多色** ➡️ 重点推荐 **NS / YH系列**。
* **升级引导（NS到YH）**：如果客户喜欢NS系列，但提到图案复杂、套印精度要求高或订单量大，务必推荐YH系列转盘印刷机：“YH系列具备微调功能和铝合金印台，虽然价格高一些，但能为您节省大量的对版和调整时间，是专业丝印车间的必备设备。”


### 常见问题
* **问：机器买回去需要插电或接气泵吗？**
  * **答**：完全不需要。这是100%纯机械手动设备，开箱组装完成后即可开始工作。
