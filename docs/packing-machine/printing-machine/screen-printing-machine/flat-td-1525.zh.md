## 四、操作与工作流程

### 1. 所需材料
操作本机器前，客户需自行准备或向我们购买以下耗材及辅助设备：
* **空压机**：必备。建议配备容量约90L、功率约2.55KW的空压机，为机器提供气动气源。
* **网版**：需根据待印刷图案定制网版。
* **油墨**：根据印刷产品的材质（如金属、玻璃等）选择相应特性的油墨。
* **稀释剂**：用于调整油墨粘度，确保印刷顺畅。
* **刮刀**：安装在机器上，用于刮压油墨。
* **洗网水**：用于印刷后清洗网版上残留的油墨。
* **治具**：用于将产品精确固定在工作台面上。

### 2. 标准工作流程
1. **安装网版**：将定制好图案的网版牢固安装到机器的网框夹具中。
2. **放置产品**：将待印刷的平面产品放置在工作台面上。
3. **添加油墨**：将适量调配好的油墨倒在网版上。
4. **开始印刷**：通过控制面板或脚踏开关启动，机器将沿导轨自动下降并完成图案印刷。
