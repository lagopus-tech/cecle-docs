## I. Product Overview

### 1. Product Category
* **Equipment Name**：Fully Automatic Flatbed Roll to Roll Screen Printing Machine
* **Model**：TD-R4060 / TD-R5080
* **Equipment Positioning**：Automated roll material flatbed printing equipment. It integrates automatic roll feeding, automatic printing, automatic drying, and automatic rewinding all into one system.
* **Form Factor**：**Production line type large-scale combined equipment**.  It consists of a feeding unit, a flatbed screen printing main machine, and an oven-rewinding unit.

### 2. Applications
* **Industry Applications**：Widely used for anti-counterfeiting trademarks, heat transfer stickers, adhesive labels, etc.
* **Material Characteristics**：Suitable for roll materials with a thickness between 0.035-0.4mm, such as PVC, OPP, polyester, and adhesive paper.
* **Production Scale**：Ideal for factory-based, mass, and highly efficient continuous production.

### 3. Core Functions
* **Multi-color Automatic Tracking & Alignment**：Utilizes a control system featuring a Japanese Mitsubishi PLC paired with a German SK color code sensor for precise and stable multi-color registration alignment.
* **Servo Drive System**：The movement of the squeegee is driven by a servo motor. The printing position and speed can be set independently, ensuring uniform ink flooding and smooth horizontal printing. This delivers clear and consistent printing results, significantly reducing color differences and missed print dots.
* **Web-pulling Device**：Employs a suction pulling head that provides uniform tension, resolving the problem of material drifting left or right during pulling. It effectively minimizes material scratches and eliminates edge deformation damage caused by pressure rollers, rendering overprinting more accurate for easily deformed materials like aluminum foil.
* **Off-grid Function**：During printing, the screen frame lifts off the substrate following the angle of the squeegee. This keeps fine characters and dots clear and prevents them from sticking to the mesh during high-speed operation, making it perfect for printing large-area background colors, glue, and other specialty inks.
* **Automatic Shutdown & Alarm**：Automatically counts the prints and triggers an alarm/stops the machine once the preset quantity is reached. It is also equipped with automatic shutdown alarms for material depletion and mechanical faults.
