## IV. Flusso di Lavoro

### Flusso di Lavoro Standard
1. **Avvio e Preriscaldamento**: Accendere l'alimentazione e avviare il ventilatore di scarico. Accendere le lampade UV una alla volta, osservare l'amperometro esterno e attendere circa 3-5 minuti fino a quando le lampade UV raggiungono uno stato di lavoro stabile.
2. **Impostazione Parametri**：Regolare la velocità del nastro trasportatore (5-25m/min) e l'altezza di irraggiamento del paralume UV in base al materiale del prodotto da polimerizzare e alle caratteristiche dell'inchiostro/adesivo UV.
3. **Alimentazione Continua**：Posizionare i prodotti stampati con inchiostro UV o con adesivo applicato sull'estremità di ingresso del nastro in Teflon. I prodotti entreranno senza problemi nella zona di polimerizzazione insieme al nastro.
4. **Raffreddamento e Raccolta**：La polimerizzazione è completa quando il prodotto esce dall'estremità di uscita, dove può essere raccolto manualmente o da un collettore automatico.
