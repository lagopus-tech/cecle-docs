## I. Обзор продукта

### 1. Категория продукта
* **Наименование оборудования**: Машина трафаретной печати с наклонным рычагом
* **Модель продукта**: TD-1518
* **Позиционирование оборудования**: Малогабаритное пневматическое оборудование для однокрасочной трафаретной печати, специально разработанное для мелких плоских изделий.
* **Форм-фактор**: **Настольное исполнение**. Вес нетто машины составляет около 50 кг. Для работы требуется размещение на прочном верстаке с достаточной несущей способностью. Машина оснащена ножной педалью, что позволяет оператору работать сидя и полностью освобождает руки.

### 2. Область применения
* **Отрасли применения**: Широко используется для однокрасочной печати на различных мелких плоских изделиях, таких как металлические листы, стеклянные линзы, деревянные панели, CD-карты, бумага и т.д.
* **Масштаб производства**: Подходит для средне- и мелкосерийного непрерывного производства, а также для коммерческой заказной серийной печати.

### 3. Основные функции
* **Интуитивно понятная панель управления**: Оснащена многокнопочной панелью управления, каждая функция управляется независимо, что делает эксплуатацию простой и удобной.
* **Пневматическая система печати**: Работа печатной головки, ракеля и валика для возврата краски осуществляется пневмоцилиндрами, что обеспечивает плавность хода и равномерность давления.
* **Механизм всесторонней регулировки**: Рабочий стол поддерживает микроподстройку, баланс и угол ракельного узла регулируются независимо.
* **Управление ножной педалью**: Ножная педаль выполняет функцию кнопки пуска. Однократное нажатие запускает цикл печати, что значительно повышает эффективность ручной загрузки/выгрузки.

## IV. Эксплуатация и рабочий процесс

### 1. Необходимые материалы
Перед началом работы на данной машине клиентам необходимо самостоятельно подготовить или приобрести у нас следующие расходные материалы и вспомогательное оборудование:
* **Компрессор**: Обязателен. Рекомендуется оснастить компрессором объемом около 90 л и мощностью около 2.55 кВт для обеспечения пневматического источника воздуха для машины.
* **Трафаретная сетка**: Необходимо изготовить заказную трафаретную сетку в соответствии с рисунком для печати.
* **Краска**: Выберите краску с соответствующими свойствами в зависимости от материала печатаемого изделия (например, металл, стекло и т.д.).
* **Растворитель**: Используется для регулировки вязкости краски, чтобы обеспечить плавную печать.
* **Ракель**: Устанавливается на машину для протягивания и прижатия краски.
* **Смывка для сетки**: Используется для очистки остатков краски с трафаретной сетки после печати.
* **Приспособление (оснастка)**: Используется для точной фиксации изделия на столе. Для некоторых изделий требуется изготовление заказных приспособлений.

### 2. Стандартный рабочий процесс
1. **Установка трафаретной сетки**: Надежно закрепите трафаретную сетку с нанесенным рисунком в зажимах рамы машины и отрегулируйте угол ракеля.
2. **Размещение изделия**: Поместите плоское изделие для печати на рабочий стол (при необходимости зафиксируйте его с помощью приспособления).
3. **Нанесение краски**: Налейте необходимое количество подготовленной краски на трафаретную сетку.
4. **Запуск печати**: Нажмите на педаль, и машина автоматически выполнит один цикл протягивания краски и печати.
