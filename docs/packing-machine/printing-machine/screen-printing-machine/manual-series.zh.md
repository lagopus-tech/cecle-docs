## 四、操作与工作流程

### 1. 所需材料
运行本机无需空压机或电源。客户只需准备基本耗材：
* **网版**：需根据印刷图案定制网版。
* **油墨**：根据待印刷的承印物材质选择合适的油墨类型。
* **其他耗材**：刮胶、洗网水、稀释剂。

### 2. 标准工作流程
1. **夹紧网版**：将网版一侧插入机器的金属框夹中，拧紧螺丝固定，并调节配重。
2. **放置产品**：抬起网版，将待印产品平放在台面上，并做好定位标记（可贴定位胶带）。
3. **手动印刷**：放下网版，在网版上倒入适量油墨，双手握住刮胶，以均匀压力向后刮墨。
4. **抬网取物**：一手抬起网版，取出印刷好的产品，完成一个印刷循环。

## 五、销售技巧与常见问题

### 销售技巧
* **精准选型指导**：客户询价时，首要问题必须是：“您要印刷的产品长宽尺寸是多少？”如果物品尺寸超过台面尺寸，必须引导客户升级到更大型号，以避免因产品放不下而产生的售后纠纷。
* **推荐(A)铸铝型号**：报价时，销售人员应重点推荐带(A)前缀的铸铝型号。可以这样引导：“只需多花10到20美元，就能获得纯铝铸造的台面，使用寿命翻倍。我们强烈推荐选择铸铝版本。”

### 常见问题
* **问：常规贴片台面和纯铝台面到底有什么区别？**
  * **答**：常规贴片台面是铝板贴面，而纯铝台面是纯铝铸造而成，更厚实，耐用性显著提升。
* **问：这台机器能在布袋上印刷吗？**
  * **答**：可以，本机适用于平面产品，适合印刷平布片或帆布袋。
* **问：机器买回来后需要插电或连接气泵吗？**
  * **答**：完全不需要。这是一台100%纯机械手动设备，收到货拧紧螺丝后即可开始工作。
