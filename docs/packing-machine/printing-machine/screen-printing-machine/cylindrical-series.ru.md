## I. Обзор продукции

### 1. Категория продукции
* **Наименование оборудования**: Машина для цилиндровой трафаретной печати
* **Модель**: TD-250 / TD-400 / TD-600 / TD-800 / TD-1100
* **Позиционирование оборудования**: Оснащенное микропроцессорным управлением и пневматическим приводом в стандартной комплектации для всей серии, это эффективное промышленное оборудование для печати по криволинейным поверхностям, специально разработанное для цилиндрических бумажных стаканчиков, стаканов для чая с молоком и т.д.

### 2. Области применения
* **Отрасли применения**: Широко используется в упаковке для общественного питания, упаковке косметики, товаров бытовой химии и других отраслях.
* **Применяемые изделия (цилиндры)**:
**Малый/Средний размер (TD-250/TD-400/TD-600)**: Тюбики для помады, пробирки, косметические баночки для крема, стаканы для чая с молоком, стеклянные бутылки.
**Большой размер (TD-800/TD-1100)**: Ведра для краски, бочки для моторного масла, ведра для воды и т.д.

### 3. Основные функции
* **Микропроцессорное управление**: Высокий уровень автоматизации, простота эксплуатации, встроенный 4-значный автоматический счетчик.
* **Автоматический ракель/заливка**: Автоматическое перемещение ракеля, заливка краски, подъем/опускание рабочего стола. Это упрощает работу, обеспечивая при этом стабильное и равномерное качество печати.
* **Всесторонняя точная настройка**: Рабочий стол можно точно регулировать по вертикали и углу наклона по горизонтали. Рамку сита можно точно регулировать вперед/назад, влево/вправо и вверх/вниз. Ракель также можно точно регулировать влево/вправо. Это обеспечивает высокую точность печати, а также удобство и быстроту настройки.

## IV. Эксплуатация и рабочий процесс

Перед началом работы на этой машине заказчику необходимо самостоятельно подготовить следующие расходные материалы и вспомогательное оборудование или приобрести их дополнительно в нашей компании:
### 1. Необходимые материалы
* **Воздушный компрессор**: Обязательный элемент. Вся серия состоит из пневматического оборудования. Заказчик должен предоставить собственный воздушный компрессор мощностью около 2,2 кВт для подачи воздуха.
* **Сетчатый трафарет**: Требуются трафареты, изготовленные на заказ в соответствии с рисунком печати.
* **Приспособление и шестерня**: Для фиксации изделия во время печати необходимо изготовить специальное приспособление. Для печати в один цвет шестерня не требуется, но для совмещения нескольких цветов / двухцветной печати необходима шестерня для точного позиционирования.
* **Расходные материалы**: Краска, растворитель, средство для промывки трафарета, ракель.

### 2. Стандартный рабочий процесс
1. **Установка трафарета**: Надежно закрепите трафарет с вашим индивидуальным дизайном в зажиме рамки сита машины.
2. **Добавление краски**: Налейте соответствующее количество подготовленной краски на трафарет.
3. **Размещение изделия**: Поместите цилиндрическое изделие для печати на опорные ролики (используя приспособление для его фиксации).
4. **Запуск печати**: Нажмите пусковую кнопку, и машина автоматически выполнит печать рисунка.
