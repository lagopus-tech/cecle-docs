## 四、工作流程

### 标准工作流程
1. **启动预热**：接通电源，启动排风机。逐一开启UV灯管，观察外部电流表，等待约3-5分钟，直至UV灯管达到稳定工作状态。
2. **参数设置**：根据待固化产品的材质及UV油墨/胶水的特性，调节输送带速度（5-25m/min）及UV灯罩的照射高度。
3. **连续送料**：将印刷好UV油墨或涂好胶水的产品放置在特氟龙网带的输入端，产品随网带平稳进入固化区域。
4. **冷却收集**：产品从输出端送出即完成固化，可由人工或自动收料机进行收集。
