## IV. Технологический процесс

### Стандартный технологический процесс
1. **Запуск и предварительный нагрев**: Включите питание и запустите вытяжной вентилятор. Включайте УФ-лампы по одной, наблюдайте за внешним амперметром и подождите примерно 3-5 минут, пока УФ-лампы не достигнут стабильного рабочего состояния.
2. **Настройка параметров**: Отрегулируйте скорость конвейерной ленты (5-25 м/мин) и высоту облучения УФ-плафона в соответствии с материалом отверждаемого изделия и характеристиками УФ-краски/клея.
3. **Непрерывная подача**: Поместите изделия, напечатанные УФ-краской или с нанесенным клеем, на входной конец тефлоновой сетки. Изделия плавно войдут в зону отверждения вместе с лентой.
4. **Охлаждение и сбор**: Отверждение завершается, когда изделие выходит с выходного конца, где его можно собрать вручную или с помощью автоматического сборщика.
