## 一、产品概述

### 1. 产品类别
* **设备名称**：高精度双伺服平面丝网印刷机
* **产品型号**：TD-2030SF
* **设备定位**：立式结构、伺服驱动的工业级高端丝网印刷设备，专为追求极致印刷精度（±0.02mm）的电子及高科技行业设计。
* **外形结构**：**落地式机型**。机箱极其稳固，配有地脚螺栓，需放置于平整的工业厂房地面运行。

### 2. 应用领域
* **行业应用**：专攻高精度印刷领域。广泛应用于PCB线路板、导电膜线路、镜片视窗、有机玻璃等。
* **材料特性**：支持最大厚度达200MM（20厘米）的基材。
* **生产规模**：适用于对良品率有严格要求的中大型高精度连续生产线。

### 3. 核心功能
* **PLC程序**：采用PLC程序控制电路及人机界面（HMI），操作人性化程度高。支持手动/半自动/全自动模式。自动印刷间隔时间采用数字控制（1~9秒）。
* **伺服系统**：印刷拖版采用伺服系统驱动，行程由光电眼控制，动作极其平稳，速度稳定可调，彻底消除气动机的"顿挫感"。
* **立式结构**：网框垂直上下升降，印刷头左右移动。油墨流动保持完美平衡，确保绝对均匀的墨层沉积。
* **真空吸附工作台**：印刷台面采用真空吸附工作台，轻松固定片材。印刷平台水平和垂直方向均配备精密微调手柄，确保精准微调，多色套印方便。

## 四、操作与工作流程

### 1. 所需物料
操作本机前，客户需另行准备或采购以下耗材及辅助设备：
* **空压机**：必须配备。需要一台空压机（提供0.5-0.7Mpa气压）为机器提供气动气源。
* **网版**：需定制高张力精密网版，以匹配机器0.02mm的精度。
* **专用油墨**：如导电银浆、阻焊油墨（绿油）、UV专用油墨等。
* **其他耗材**：稀释剂、刮刀、洗网水、治具。

### 2. 标准工作流程
1. **精密对位**：将网框安装至夹臂，利用工作台下方的精密微调手柄进行精确的X/Y轴对位。
2. **参数设置**：在PLC触摸屏上设置印刷行程、刮刀速度、回墨刀速度及全自动间隔时间。
3. **放置产品**：将产品放置在工作台上，开启真空吸附功能将其牢固固定。
4. **开始印刷**：伺服电机驱动刮刀平稳滑过，完成极高精度的图案转移。

## 五、销售技巧与常见问题

### 销售技巧
* **卖点强调**：向客户重点强调"伺服电机"和"立式结构"。这是其区别于低端气动机的关键。气动机速度快但有震动，而伺服电机确保刮刀以恒定、平稳的速度在网版上滑行——这是印刷精密电路板的先决条件。
* **定制服务**：在外贸沟通中，主动提及机器可定制110V或220V电压，能显著提高转化率。
* **捆绑销售**：收到询盘时，主动询问客户是否需要定制网版，提供"一站式解决方案"以增加客户粘性。

### 常见问题
* **问：为什么这台机器比普通平面丝印机贵？**
  * **答**：因为其驱动核心完全不同。普通机器使用气动气缸推拉，而本机采用伺服系统配合PLC程序控制。其印刷精度达到行业领先的±0.02mm，专为制造精密电子元件和电路板而设计。
* **问：立式结构有什么好处？**
  * **答**：普通铰链式（斜臂）丝印机的网版是倾斜抬起的，油墨容易回流。而立式结构的网版是水平垂直升降的。这意味着油墨始终均匀分布在网版上，这是对墨层厚度一致性有严格要求的电子产品印刷所必需的。
