## 一、产品概述

### 1. 产品类别
* **设备名称**：全自动平网卷对卷丝网印刷机
* **型号**：TD-R4060 / TD-R5080
* **设备定位**：自动化卷材平网印刷设备。集自动送料、自动印刷、自动烘干、自动收卷于一体。
* **产品形态**：**生产线式大型组合设备**。由放料单元、平网丝印主机、烘箱收卷单元组成。

### 2. 应用范围
* **行业应用**：广泛应用于防伪商标、热转印贴纸、不干胶标签等。
* **材料特性**：适用于厚度在0.035-0.4mm之间的卷材，如PVC、OPP、聚酯、不干胶纸等。
* **生产规模**：适合工厂化、大批量、高效率的连续生产。

### 3. 核心功能
* **多色自动追踪对位**：采用日本三菱PLC搭配德国SK色标传感器的控制系统，实现精准稳定的多色套印对位。
* **伺服驱动系统**：刮刀运动由伺服电机驱动。印刷位置和速度可独立设定，确保均匀的覆墨和平稳的水平印刷。实现清晰一致的印刷效果，显著减少色差和漏印点。
* **拉料装置**：采用吸气拉料头，提供均匀张力，解决拉料过程中材料左右跑偏的问题。有效减少材料划伤，消除压辊造成的边缘变形损伤，使铝箔等易变形材料的套印更加精准。
* **离网功能**：印刷时，网框跟随刮刀角度抬起脱离承印物。使精细文字和网点保持清晰，防止高速运行时粘网，非常适合印刷大面积底色、胶水等特殊油墨。
* **自动停机报警**：自动计数印刷数量，达到预设数量后触发报警/停机。同时配备缺料和机械故障自动停机报警功能。

* **高效集成**：相比半自动设备，可实现从卷料到成品的单人操作，大幅节省人工成本。

## 四、操作与要求

* **场地准备**：因整机线体较长，需预留足够的直线空间进行安装。
* **空压机**：必备设备。机器需连接空压机才能运行。
* **网版**：需根据印刷图案定制网版。
* **必备耗材**：专用卷材印刷油墨、洗网水、开油水、刮胶等。
