## 四、操作与工作流程

### 1. 所需物料
操作本机器前，客户需自行准备或向我方购买以下耗材及辅助设备：
* **空压机**：必备。建议配备容量约90L、功率约2.55KW的空压机，为机器提供气动气源。
* **网版**：需根据待印刷图案定制网版。
* **油墨**：根据印刷产品材质（如金属、玻璃等）选择相应特性的油墨。
* **稀释剂**：用于调整油墨粘度，确保印刷顺畅。
* **刮刀**：安装于机器上，用于刮压油墨。
* **洗网水**：用于印刷后清洗网版上残留的油墨。
* **治具**：用于将产品精准固定于台面上。部分产品需定制专用治具。

### 2. 标准工作流程
1. **安装网版**：将定制好图案的网版牢固安装至机器的网框夹具中，并调整刮刀角度。
2. **放置产品**：将待印刷的平面产品放置于台面上（根据需要可使用治具固定）。
3. **添加油墨**：将适量调配好的油墨倒入网版上。
4. **启动印刷**：按下开关，机器将自动完成一次刮墨印刷动作。
