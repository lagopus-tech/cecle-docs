## IV. Эксплуатация и рабочий процесс

### 1. Необходимые материалы
Для работы этого станка не требуется компрессор или источник питания. Клиентам необходимо подготовить только основные расходные материалы:
* **Трафаретная сетка**: Требуются сетки, изготовленные на заказ в соответствии с печатаемыми рисунками.
* **Краска**: Выберите подходящий тип краски в зависимости от материала подложки.
* **Прочие расходные материалы**: Ракель, средство для промывки сеток, разбавитель.

### 2. Стандартный рабочий процесс
1.  **Зажим сетки**: Вставьте одну сторону сетки в металлические зажимы рамы станка, затяните винты для фиксации и отрегулируйте противовес.
2.  **Размещение изделия**: Поднимите сетку, поместите изделие для печати ровно на стол и сделайте метки позиционирования (можно нанести позиционирующую ленту).
3.  **Ручная печать**: Опустите сетку, нанесите необходимое количество краски на сетку, возьмите ракель обеими руками и с равномерным нажимом протяните краску назад.
4.  **Подъем сетки и извлечение подложки**: Поднимите сетку одной рукой и извлеките напечатанное изделие, завершив один цикл печати.

  * **О**: Да, этот станок может печатать на плоских изделиях, что делает его пригодным для печати на плоских тканевых листах или холщовых сумках.
