## 一、产品概述

### 1. 产品类别
* **设备名称**：半自动立式平面丝网印刷机
* **型号**：TD-3050 / TD-4060 / TD-5080 / TD-60100
* **设备定位**：一种立式升降丝网印刷设备。采用网框垂直升降、印刷头左右移动的立式结构。
* **外形结构**：**立式机型**。运行极其稳定，适合工厂高强度平面印刷任务。

### 2. 应用领域
* **行业应用**：广泛应用于玻璃面板、金属板材、标牌、纸张、陶瓷贴花及各类平面材料的丝网印刷。
* **材料特性**：立式升降结构有效防止油墨流淌，确保运行稳定。非常适合厚度达30mm的平面产品。
* **生产规模**：适用于中大型工厂的连续生产加工，最高印刷速度可达1000次/小时。

### 3. 核心功能
* **立式升降结构**：网框垂直平稳升降，印刷头由变频电机驱动左右移动，确保印刷精度和涂墨均匀。
* **三种模式切换**：支持自动、半自动和手动模式，适应不同熟练程度的操作人员。
* **真空装置**：工作台配备真空吸附装置，并支持前后、左右微调，便于快速套准。

## 二、核心优势

* **立式结构**：垂直升降确保油墨在网版上分布更均匀，防止因倾斜导致的漏墨或重影。
* **强劲变频驱动**：印刷头左右移动由变频电机驱动，左右运行速度可调，刮刀压力极其恒定。
* **人性化设计**：刮刀/回墨刀及网版的安装、拆卸和清洁简便易行。

## 四、操作与工作流程

### 1. 所需辅助设备
操作本机前，客户需另行准备或购买以下耗材及辅助设备：
* **空压机**：必备。机器需连接空压机才能运行。
* **网版**：需根据印刷图案定制网版。
* **油墨**：需根据承印物材质选择相应特性的油墨。
* **耗材准备**：稀释剂、刮刀、洗网水、治具。

### 2. 标准工作流程
1. **机器设置**：安装网版，调整刮刀压力和印刷行程。
2. **上料**：手动将承印物放置在工作台上。
3. **印刷**：触发脚踏开关或设定自动循环，机器完成印刷周期。
