## 一、产品概述

### 1. 产品类别
* **设备名称**：圆面丝印机
* **型号**：TD-250 / TD-400 / TD-600 / TD-800 / TD-1100
* **设备定位**：全系列标配微电脑控制与气动驱动，专为圆柱形纸杯、奶茶杯等设计的高效工业级曲面印刷设备。

### 2. 应用领域
* **行业应用**：广泛应用于餐饮包装、化妆品包装、日化产品等行业。
* **适用产品（圆柱体）**：
**中小型（TD-250/TD-400/TD-600）**：口红管、试管、化妆品膏霜瓶、奶茶杯、玻璃瓶。
**大型（TD-800/TD-1100）**：油漆桶、机油桶、水桶等。

### 3. 核心功能
* **微电脑控制**：自动化程度高，操作简单，内置四位自动计数器。
* **自动刮墨/回墨**：自动刮墨、回墨及工作台升降。简化操作，同时确保稳定一致的印刷质量。
* **全方位微调**：工作台可上下及水平角度微调。网框可前后、左右、上下微调。刮刀亦可左右微调。实现高印刷精度，同时调节方便快捷。

## 四、操作与工作流程

在运行本机前，客户需自行准备以下耗材及辅助设备，或向我司另行购买：
### 1. 所需材料
* **空压机**：必备品。全系列为气动设备，客户需自备功率约2.2KW的空压机提供气源。
* **网版**：需根据印刷图案定制网版。
* **治具与齿轮**：需根据产品制作专用治具以固定印刷位置。单色印刷无需齿轮，但多色套印/双色印刷需使用齿轮进行精准定位。
* **耗材**：油墨、开油水、洗网水、刮刀。

### 2. 标准工作流程
1. **安装网版**：将印有您定制图案的网版牢固地安装到机器的网框夹具中。
2. **添加油墨**：将适量调配好的油墨倒在网版上。
3. **放置产品**：将待印刷的曲面产品放置在支撑滚轮上（使用治具固定）。
4. **开始印刷**：按下启动开关，机器将自动完成图案印刷。

### 常见问题
* **问：这台机器可以印刷多色（多色套印）吗？**
  * **答**：丝印机主要设计用于单色印刷。如果需要在同一个瓶子上印刷2或3种颜色，是可以的，但必须分次印刷——每次只能印刷一种颜色。关键前提是您的瓶子必须有定位点。
* **问：购买机器后还需要购买什么？**
  * **答**：这是一台气动机器，所以您必须购买空压机。此外，需要由我们为您特定的瓶子定制专用治具。您还需要准备网版、油墨、刮刀、洗网水和开油水等耗材。
* **问：保修和售后服务如何？**
