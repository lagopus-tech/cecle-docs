## 一、产品概述

### 1. 产品类别
* **设备名称**：平面丝网印刷机
* **产品型号**：TD-2533

* **外形规格**：**中型桌面机**。机器净重约90KG，建议放置在承重足够、符合人体工学高度要求的坚固工作台上，避免操作人员蹲姿作业；也可直接在地面操作。

### 2. 应用范围
* **行业应用**：广泛应用于包装、电子、广告等行业的平面印刷（如玻璃、金属板、无纺布袋等）。

### 3. 核心功能
* **全新升级主控板**：运动指令响应达微秒级，性能稳定可靠。
* **精密铣削加工**：90%以上零件采用电镀/烤漆工艺，精度与耐用性稳步提升。
* **真空吸附台面**：台面布满微孔，通过真空发生器产生吸力，牢固吸附薄膜或轻薄产品，有效防止印刷后粘网。
* **气动印刷系统**：依靠5-7BAR气压驱动，摩擦阻力小，刮刀压力均匀，印刷效率高达1200件/小时。

## 四、操作与工作流程

### 1. 所需材料
操作本机前，客户需自行准备或从我司购买以下耗材及辅助设备：
* **空压机**：必备。建议配备容量100L以上、功率3KW以上的空压机，为机器提供气动气源。
* **网版**：需根据待印刷图案定制网版。
* **油墨**：根据承印物材质（如金属、玻璃等）选择相应特性的油墨。
* **稀释剂**：用于调节油墨粘度，确保印刷顺畅。
* **刮刀**：安装在机器上，用于刮压油墨。
* **洗网水**：用于印刷后清洗网版上残留的油墨。
* **治具**：用于将产品精确固定在台面上。部分产品需定制治具。

### 2. 标准工作流程
1. **安装网版**：将定制好图案的网版牢固安装到机器的网框夹具中。
2. **放置产品**：将待印刷的平面产品放置到台面上（根据需要，使用治具固定或开启真空吸附）。
3. **添加油墨**：将适量调配好的油墨倒在网版上。
4. **启动印刷**：按下启动开关，机器将完成图案印刷。
