## IV. Эксплуатация и рабочий процесс

### 1. Необходимые материалы
Перед началом работы с этой машиной клиентам необходимо самостоятельно подготовить или приобрести у нас следующие расходные материалы и вспомогательное оборудование:
* **Воздушный компрессор**: Обязателен. Рекомендуется оснастить компрессором емкостью 100 л или более и мощностью 3 кВт или более для обеспечения пневматического источника воздуха для машины.
* **Трафаретная сетка**: Необходимо изготовить индивидуальную трафаретную сетку в соответствии с рисунком для печати.
* **Краска**: Выберите краску с соответствующими свойствами в зависимости от материала печатного изделия (например, металл, стекло и т.д.).
* **Растворитель**: Используется для регулировки вязкости краски, чтобы обеспечить плавную печать.
* **Ракель**: Устанавливается на машине для протягивания и прижатия краски.
* **Средство для промывки сеток**: Используется для очистки остатков краски с сетки после печати.
* **Приспособление**: Используется для точной фиксации изделия на столе. Для некоторых изделий требуются изготовленные на заказ приспособления.

### 2. Стандартный рабочий процесс
1. **Установите сетку**: Надежно закрепите сетку с индивидуальным рисунком в зажимах рамы сетки машины.
2. **Разместите изделие**: Поместите плоское изделие для печати на столешницу (при необходимости используйте приспособления для фиксации или включите вакуумный отсос).
3. **Добавьте краску**: Налейте необходимое количество подготовленной краски на сетку.
4. **Начните печать**: Нажмите пусковой переключатель, и машина выполнит печать рисунка.
