**Flux de Travail** :
1. **Alimentation Automatique** : Placez manuellement les plateaux contenant les aliments sur le poste de travail.
2. **Positionnement du Film** : La table rotative envoie le plateau vers la zone de recouvrement du film skin, qui est appliqué automatiquement.
3. **Mise Sous Vide Skin** : Mettez sous vide et chauffez le film pour qu'il adhère fermement aux aliments.
4. **Coupe & Scellage** : Après la liaison, effectuez la coupe du film tout en scellant.
5. **Déchargement** : Après le scellage, la machine se réinitialise automatiquement et l'opérateur fait tourner manuellement la table pour retirer le produit fini.
