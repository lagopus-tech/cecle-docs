**工作流程**：
1. **自动进料**：人工将装有食品的托盘放入工位。
2. **定位覆膜**：旋转台将托盘送至贴体膜覆盖区域，自动覆膜。
3. **真空贴体**：抽真空并加热薄膜，使其紧密贴合食品。
4. **切割封口**：贴合完成后，完成切膜并封口。
5. **下料**：封口后自动复位，操作员手动旋转台面取出成品。
