**Рабочий процесс**:
1. **Автоматическая подача**: Вручную разместите лотки с продуктом на рабочей станции.
2. **Позиционирование пленки**: Поворотный стол подает лоток в зону нанесения скин-пленки, которая наносится автоматически.
3. **Вакуумная скин-упаковка**: Вакуумирование и нагрев пленки для ее плотного прилегания к продукту.
4. **Резка и запечатывание**: После склеивания выполняется резка пленки одновременно с запечатыванием.
5. **Выгрузка**: После запечатывания происходит автоматический сброс, и оператор вручную поворачивает стол для удаления готового продукта.
