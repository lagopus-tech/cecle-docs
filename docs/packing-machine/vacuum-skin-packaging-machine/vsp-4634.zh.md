### 7. 工作流程
1. **人工放料**：将装有食品的托盘放入上料区的模具腔中。
2. **自动进料**：按下启动按钮或脚踏开关，模具自动滑入机器内部。
3. **加热贴体**：上膜被加热软化，真空泵启动抽取腔室内空气，使薄膜紧密贴合食品表面。
4. **封口切膜**：贴合后，机器进行热封并沿边缘修整薄膜，确保切口整洁。
5. **下料**：模具自动滑出，操作员取出成品托盘。
