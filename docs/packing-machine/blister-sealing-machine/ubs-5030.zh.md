## 2. 核心优势
1.  **三工位旋转循环结构**：三工位布局，配备三套模具，分别用于上料、预热、封口及下料。无停机时间，连续批量生产，生产连贯性强。
2.  **压板全面积加热**：上压板加热（平面加热），配合铝模，适用于**不规则异形边缘封口产品**，不受等宽边缘限制；是异形泡壳封口的首选型号（相比模具加热，变形略大）。
3.  **有效封口平台 300×500mm (11.81×19.69英寸)**：可定制1/2/3/4腔模具，支持单个大型异形产品包装或多个小产品同时封口。
4.  **0.24T 轻量气动压合**：适中的0.24吨压力，避免压破薄壁PET泡壳和柔软的特卫强；对薄壁异形产品友好，封口牢固。
5.  **紧凑稳定机身**：占地面积小，车间摆放灵活，长期运行振动低。
6.  **全球宽电压适配**：兼容110V/220V，50~60Hz，适用于国内工厂、东南亚、欧美车间。
7.  **全面定制化**：封口尺寸、模具腔数、加热规格均可定制；一机适应多种不规则异形产品。

## 4. 操作流程
1.  在空工位手动将产品装入泡壳，将泡壳放入模具槽内并盖上纸卡/特卫强；
2.  手动旋转转盘，将已装料的模具转至压板加热工位，进行恒温预热，表面均匀受热；
3.  旋转工位至封口区域，气缸下压完成气动热封，通过全面积加热适应产品边缘；
4.  转盘继续旋转至下料工位，手动取出已封口产品，同时在空闲工位装入新物料；
5.  三套模具交替进行上料、预热-封口、下料，实现无缝全流程循环，不间断生产。

### 销售建议
1.  **目标客户**：异形/不规则产品批量生产工厂；对于常规等宽边缘封口产品，推荐BS-5030、BS-3180模具加热型号。
2.  **多品类混合生产**：定制2~4腔模具，可同时生产多种小型异形产品，提高单机产能。
3.  **选型重点**：异形泡壳、非标医用特卫强纸卡、薄壁软泡壳封口，首选UBS-5030压板加热型号。
4.  **配套设备**：客户需自备0.4~0.8MPa空压机用于生产；若无气源需购买空压机。

### 常见问题
**Q1: 是否标配3套模具？**
A1: 三工位结构需要3套模具才能实现不间断循环生产。模具不包含在主机价格内，需单独定制。

**Q2: 如何选择电压？**
A2: 可根据要求定制110V或220V。

**Q3: 是否支持多腔模具？**
A3: 有效封口面积为300×500mm (11.81×19.69英寸)，可根据产品尺寸灵活定制1/2/3/4腔模具。

**Q4: 生产效率如何？**
A4: 标准速度10循环/分钟，三工位循环操作，适用于异形产品的精细封口，适合小批量到大批量的流水线生产。

**Q5: 必须使用空压机吗？**
A5: 本机采用气动压合结构，0.4~0.8MPa空压机是必要的配套设备；无气源则无法完成封口。
