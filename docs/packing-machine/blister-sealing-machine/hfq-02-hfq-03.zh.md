> **核心摘要：**
> HFQ-02 和 HFQ-03 系列是台式手动高频吸塑焊接机，分别采用两工位推拉方台和三工位转盘结构。它们是研发打样实验室、中小型包装厂和皮革制品厂理想的熔接解决方案。其核心优势包括广泛的材料兼容性（如 APET、PETG、PVC、EVA 和皮革）、紧凑的占地面积，以及支持定制 1-4 腔模具，最大加工面积为 350×200mm。提供气动和液压两种加压方式可选，HFQ-02 擅长小批量打样，而 HFQ-03 可实现连续循环生产，产量高达每分钟 3-4 件，确保封边光滑无瑕。

## 1. 产品概述
HFQ-02 和 HFQ-03 属于同系列台式手动高频热合设备，共享相同的加热、加压和温控系统。两者仅在工作台结构上有所不同：HFQ-02 采用 **两工位推拉方台**，而 HFQ-03 配备 **三工位转盘工作台**。

两款机型均配备 200×350mm 加热板，最大加工尺寸为 350×200mm。支持定制 1~4 腔模具。温度可在 0℃ 至 300℃ 之间调节，可选气动或液压加压。重型机架重达 350KG（771.62lbs）。

适用于 APET、PETG、GAG、PVC 和 PP 材质制成的双泡壳包装、泡壳纸卡封口、牙刷、小五金和玩具包装。也可热合 EVA 织物、皮革、OPP 和 PU 材料，并能匹配小型卷料辅助生产线。
HFQ-02 两工位方台是研发打样和小批量零散订单的理想选择；HFQ-03 三工位转盘可由一名操作员实现连续循环生产，广泛应用于样品车间、中小型包装厂和皮革加工厂。

## 4. 操作流程
### HFQ-02 手动两工位方台操作步骤
1. 在两个工位上安装相同的定制模具。气动型连接空压机，液压型连接液压回路，调整下压行程和保压时间。
2. 根据材料在控制面板上设定相应的加热温度，预热机器，通过空推拉动作测试压合动作。
3. 将泡壳、纸卡和皮革工件放入外侧空闲工位的模具定位槽中。
4. 将已上料的工位推入加热区，机器自动下压进行恒温热合。
5. 封合后拉出工作台取出成品；交替在另一工位上料，实现循环操作。
6. 生产结束后关闭加热、气源或液压系统。待完全冷却后拆下模具并清理塑料残留物，定期维护滑轨。

### HFQ-03 手动三工位转盘操作步骤
1. 在转盘三个工位上安装相同的模具。气动型连接空压机，液压型连接液压回路，调整压合行程和保压时间。
2. 在温控面板上设定目标温度并预热。空载旋转转盘，检查旋转顺畅度和压合性能。
3. 将泡壳、纸卡和皮革件放入外侧空闲工位的模具定位槽中。
4. 手动旋转转盘，将已上料工位移至加热区，进行自动恒温热封。
5. 封合后旋转转盘至取料位取出成品；在空闲工位继续上料，实现不间断循环生产。
6. 工作结束后关闭加热、气源和液压系统。冷却后拆卸模具并清理残留物；定期润滑转盘轴承和加热组件。

### 常见问题
问：两台机器都需要空压机吗？
答：气动版本需要 0.4~0.8MPa 的空压机；液压版本无需气源即可工作。

问：HFQ-02 和 HFQ-03 的模具可以互换吗？
答：可以，两者加热板尺寸相同，因此模具可以互换使用。

问：最高加热温度是多少？能否熔接真皮和 EVA 织物？
答：温度调节范围为 0~300℃。通过适当的参数设置，可以实现 PU、真皮和 EVA 布料的热熔。

问：允许的最大工件尺寸是多少？
答：工件尺寸不得超过 350×200mm（13.78''×7.87''；更大的产品需要更大的机器）。

问：两款机型的保修政策相同吗？
答：是的。整机保修 1 年，不含耗材，提供终身技术支持。
