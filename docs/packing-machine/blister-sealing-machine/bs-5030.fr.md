## 4. Processus de fonctionnement
1. La table rotative contient trois postes. Les opérateurs placent les produits dans les blisters au niveau du moule du poste vide, mettent les blisters dans les rainures du moule, puis recouvrent avec une carte papier ou du papier Tyvek.
2. Faites tourner la table rotative pour déplacer le moule chargé vers le poste de chauffage pour le préchauffage.
3. Faites tourner le poste vers la zone de scellage, le vérin pneumatique appuie pour terminer le scellage thermique pneumatique.
4. Continuez à faire tourner la table vers le poste de déchargement ; les opérateurs retirent les produits finis et remplissent les matériaux en même temps pour un fonctionnement cyclique.
5. Trois jeux de moules travaillent en alternance pour l'alimentation, le scellage thermique et le déchargement afin de réaliser une production continue ininterrompue.

### Conseils de vente
1. Ciblez les clients ayant des demandes de production de masse ; recommandez des modèles de table de petite taille pour la réalisation d'échantillons en petits lots
2. Pour les clients ayant des commandes de produits diversifiées, recommandez des moules multi-empreintes personnalisés de 2 à 4 cavités pour améliorer le rendement d'une seule machine
3. Privilégiez la BS-5030 pour les scénarios nécessitant le scellage de blisters PET épais et de cartes papier Tyvek de qualité médicale
4. Les clients disposant de leur propre compresseur d'air peuvent mettre la machine en production directement ; les clients sans source d'air doivent acheter un compresseur d'air supplémentaire

### Questions fréquentes
**Q1 : Est-il obligatoire d'équiper la machine de 3 jeux de moules ?**
R1 : La machine adopte une structure de table rotative à 3 postes. 3 jeux de moules sont normalement requis pour réaliser une production cyclique ininterrompue. Les moules sont personnalisés séparément avec des frais supplémentaires.

**Q2 : Comment sélectionner la tension ?**
R2 : Elle prend en charge l'alimentation monophasée 110 V/220 V. Le 110 V ou 220 V peut être personnalisé pour les commandes de commerce extérieur selon les besoins.

**Q3 : Les moules multi-empreintes peuvent-ils être personnalisés ?**
R3 : La surface de scellage est de 500 × 300 mm (19,69 × 11,81 pouces), et des moules à 1/2/3/4 empreintes peuvent être personnalisés sur demande.

**Q4 : Quelle est l'efficacité de production ?**
R4 : 15 à 20 cycles de scellage par minute, adapté à la production de masse à grande échelle.

**Q5 : Un compresseur d'air est-il nécessaire ?**
R5 : La machine adopte une structure de pressage pneumatique, un compresseur d'air avec une source d'air de 0,4 à 0,8 MPa est un équipement de support essentiel.
