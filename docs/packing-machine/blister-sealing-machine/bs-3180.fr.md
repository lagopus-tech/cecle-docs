## 4. Procédure de Fonctionnement
1. La table rotative contient trois postes. Les opérateurs placent les produits dans les blisters sur le moule du poste vide, insèrent les blisters dans les rainures du moule, puis couvrent avec une carte papier ou une feuille de Tyvek.
2. Faites tourner la table pour déplacer le moule chargé vers le poste de chauffage pour le préchauffage.
3. Faites tourner le poste vers la zone de scellage, le vérin pneumatique descend pour effectuer le scellage par compression thermique pneumatique.
4. Continuez à faire tourner la table vers le poste de déchargement ; les opérateurs retirent les produits finis et rechargent les matériaux simultanément pour un fonctionnement cyclique.
5. Les trois jeux de moules alternent pour l'alimentation, le thermoscellage et le déchargement afin d'obtenir une production continue ininterrompue.

### Foire Aux Questions
**Q1 : Les 3 jeux de moules sont-ils obligatoires pour la machine ?**
R1 : La machine adopte une conception rotative à 3 postes. 3 jeux de moules sont requis en standard pour réaliser une production cyclique ininterrompue ; les moules sont personnalisés et facturés séparément.

**Q2 : Comment sélectionner la tension correcte ?**
R2 : L'alimentation monophasée 110V/220V est prise en charge ; une configuration personnalisée 110V/220V est disponible pour les commandes à l'exportation.

**Q3 : Les moules multi-cavités peuvent-ils être personnalisés ?**
R3 : La surface de scellage mesure 540*380 mm (21,26×14,96 pouces) ; des moules à 1/2/3/4 cavités peuvent être personnalisés sur demande.

**Q4 : Quelle est l'efficacité de production ?**
R4 : 10 cycles de scellage par minute, avec un rendement plusieurs fois supérieur au modèle de bureau BS-2030, idéal pour la production de masse à grande échelle.

**Q5 : Un compresseur d'air est-il nécessaire ?**
R5 : La machine adopte une structure de compression pneumatique ; un compresseur d'air avec une source d'air de 0,4~0,8 MPa est un équipement de support essentiel.
