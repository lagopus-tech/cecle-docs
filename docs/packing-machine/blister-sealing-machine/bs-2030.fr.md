## 4. Processus de Fonctionnement
1. Placer manuellement le produit dans le blister.
2. Mettre le blister avec le produit sur la plaque de moule et recouvrir avec la carte en papier.
3. Pousser manuellement la plaque de moule dans le poste de scellage.
4. La machine appuie automatiquement de manière pneumatique pour effectuer le thermoscellage.
5. Après le scellage, tirer manuellement la plaque de moule et retirer le produit fini.

### FAQ
**Q1 : La machine nécessite-t-elle un compresseur d'air ?**
R1 : Oui, c'est une machine pneumatique. Elle doit être connectée à un compresseur d'air de 0,6–0,8 MPa pour fonctionner correctement.

**Q2 : Le moule est-il inclus ?**
R2 : Non, le moule n'est pas inclus et doit être personnalisé séparément.

**Q3 : La température est-elle réglable ?**
R3 : Oui, elle prend en charge le réglage numérique de la température en fonction des différents matériaux.

**Q4 : Est-ce difficile à utiliser ?**
R4 : Très facile. Il suffit de pousser/tirer la plaque de moule manuellement ; la machine scelle automatiquement. Vous pouvez la maîtriser en quelques minutes.

**Q5 : Quels matériaux d'emballage sont adaptés ?**
R5 : Spécialement utilisé pour le thermoscellage de blisters PVC / PET + cartes en papier.
