## 1. 产品概述
HFQ-01 单头滑台式高频泡壳焊接机是一款推拉式工作台热封设备。适用于 APET、PETG 和 GAG 材料的环保型双泡壳封合、泡壳纸卡包装以及牙刷封口包装。它还支持 EVA 织物、皮革、OPP 和 PU 材料的热粘合，并可匹配 APET、PETG、PVC 和 PP 柔性卷材生产线。

采用双工位推拉滑台结构，配备 300×400mm 加热板，最大加工尺寸达 400×300mm。可定制 1~4 腔模具。支持气动和液压两种加压模式，温度可在 0℃ 至 300℃ 之间调节。广泛应用于包装厂、塑料加工厂和皮革加工企业，用于日用品、塑料制品和皮革制品的标准化批量包装。

## 2. 核心优势
- 双工位推拉滑台设计实现同步送料与封合，支持不停机连续生产，大幅提升单机产量
- 大尺寸热封台：300×400mm（11.81''×15.75''），兼容 1/2/3/4 腔定制模具，可一次加工多个产品
- 可选配气动和液压加压模式；液压版本压力均匀，可避免厚泡壳和多层皮革封合不足及边缘脱开
- 0~300℃ 全范围温控系统，兼容塑料、皮革、织物等多种不同熔点材料，实现一机多用
- 重型机架结构，净重 350KG（771.62lbs），长期连续生产运行稳定无抖动，封边光滑
- 宽电压兼容 220V/380V，50~60Hz，适应国内外工厂供电条件
- 结构简单，易于维护；滑轨和加热组件可拆卸，便于日常清洁保养
- 可连接柔性卷材生产线，适用于独立单机生产和流水线配套作业

## 4. 操作流程
1. 根据产品尺寸安装定制模具。气动型连接空压机，液压型连接液压油路，并调试滑台行程；
2. 根据加工材料在温控面板上设定封合温度，开启加热预热，空运行滑台检查加压性能；
3. 拉出空闲滑台，将泡壳、纸卡及皮革工件整齐放入模具定位槽内，不得偏移；
4. 将装载好的滑台推入机器腔体，设备自动下压进行恒温热压，完成泡壳封合和材料焊接；
5. 实现交替循环生产：一个工位进行封合作业时，另一工位完成上料装填；
6. 生产结束后关闭加热系统、气源和液压系统。待完全冷却后拆下模具并清理塑料残留物，完成设备日常保养。

问：操作时是否必须配备空压机？
答：气动加压版本需要空压机（0.4~0.8MPa），液压版本则无需气源。

问：可以定制多腔模具吗？
答：可以。我们支持根据产品尺寸定制 1~4 腔模具，以提高单次产量。

问：最高加热温度是多少？能焊接皮革吗？
答：温度可在 0℃ 至 300℃ 之间调节。通过正确设置参数，可实现 EVA、PU 和真皮的热粘合。

问：最大加工尺寸是多少？
答：产品尺寸不得超过 400×300mm（15.75''×11.81''）。更大尺寸的产品需要升级型号。
