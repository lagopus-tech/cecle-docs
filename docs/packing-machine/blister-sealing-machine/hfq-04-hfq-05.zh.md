# HFQ-04 / HFQ-05 全自动高频泡壳焊接机

> HFQ-04和HFQ-05系列是全自动转盘式高频泡壳焊接机，适用于大型包装厂、电子配件制造商和医疗泡壳工厂。这些机器擅长对APET、PETG、PVC和皮革材料进行不间断高速热封合，无需使用泡壳油。核心优势在于其先进的PLC触摸屏控制和3工位自动旋转工作台，可将生产效率显著提升至6-10件/分钟。基础型HFQ-04提供可靠的自动旋转和热封功能，而升级版HFQ-05配备伺服机械臂，可实现完全无人值守的自动卸料，完美融入24小时连续生产线，降低人工成本。

## 1. 产品概述
HFQ-04和HFQ-05是全自动转盘式高频热合机。它们配备集成式高频振荡系统和PLC触摸屏自动控制系统，以及3工位自动旋转工作台。这些机器适用于APET、PETG、GAG、PVC和PP双泡壳、泡壳纸卡包装、牙刷和小五金件的连续包装。它们还可以热熔EVA、皮革、PU和OPP柔性材料，并可连接至自动化生产线。

两款机型共享相同的基础配置、尺寸和额定功率。核心区别在于物料处理方式：HFQ-04是基础自动型，需人工上下料；而HFQ-05升级配备伺服机械臂，可自动拾取和放置成品，实现无人值守连续生产。机器重量为500KG（1102.31磅），配备200×350mm热压板，最大加工尺寸为350×200mm。支持1至4腔定制模具，温度调节范围精确为0℃至300℃，非常适合中大型包装工厂的大批量不间断生产。

## 4. 操作流程
### HFQ-04 基础自动型操作步骤
1. 在三个转盘工位上安装相同的定制模具，连接0.6~0.8MPa空压机和电源。
2. 在触摸屏上设置高频功率、封合温度、转盘速度和保压时间，然后保存生产程序。
3. 预热高频发生器和热压板，运行空载自动循环测试，检查火花保护和压合性能。
4. 将泡壳、纸卡和皮革工件手动放入外部上料工位的定位槽中。
5. 机器自动将工位旋转至热封区进行恒温高频熔合，完成后旋转至卸料工位。
6. 手动取出成品并持续上料，实现不间断循环生产。
7. 生产结束后关闭高频系统、加热系统和气源。待模具完全冷却后清洁塑料残留物，并定期维护转盘分度器。

### HFQ-05 机械臂自动型操作步骤
1. 在转盘工位上安装定制模具，连接气源和电源，并在触摸屏上同步设置机械臂行程和速度参数。
2. 调整高频功率、温度和转盘循环，保存集成的全自动运行程序。
3. 预热机器并运行空载全自动循环，测试转盘、压合和机械臂取料的联动。
4. 仅需在上料工位手动放入工件；无需人工卸料。
5. 转盘自动将工件输送至热封工位进行高频熔合。旋转至卸料位置后，伺服机械臂自动抓取成品。
6. 可连接输送线将成品输送至下一工序，支持24小时无人值守循环批量生产。
7. 工作结束后关闭高频和加热系统。冷却后拆卸并清洁模具；定期润滑机械导轨和转盘轴承。

### 常见问题
问：两款机器都需要空压机吗？
答：是的。两者均采用气动压合结构，正常工作需要0.6~0.8MPa的稳定气源。

问：HFQ-04和HFQ-05的模具可以互换吗？
答：可以。它们共享相同的热压板尺寸，因此配套模具可以互换使用。

问：泡壳纸卡包装需要泡壳油吗？
答：不需要。内置高频振荡器实现材料分子自发热熔合，无需泡壳油，封合强度更高。

问：HFQ-04后期可以加装机械臂吗？
答：机械臂是HFQ-05的专属集成结构，出厂后无法安装在HFQ-04上。

问：机器保修期多长？
答：统一整机保修1年，不含易损件；提供终身技术支持。
