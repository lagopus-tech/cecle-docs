## 1. 产品概述
DPP-140/250E 是一款双功能自动平板式泡罩包装机。通过更换模具，可在**铝铝冷成型**和**铝塑热成型**模式之间切换。适用于片剂、硬胶囊、软胶囊、丸剂、保健品、小型五金件等产品的连续自动包装。

该设备集成了完整的工艺流程：基材送料、预热、成型、振动加料、在线缺粒检测、热封打码、撕裂线压痕、色标对版、冲裁、废料收卷和自动剔除。

配备三菱PLC + 威纶通触摸屏集成控制系统，编码器精确定时控制，步进电机牵引定位，所有工位均采用手轮调节+螺栓锁紧，以及快速定位模具，大幅提高换模效率。符合制药GMP标准。提供140mm（窄幅）和250mm（宽幅）两种规格，适用于中小批量多品种生产以及中高产能标准化生产线。广泛应用于制药厂、保健品生产企业、医疗器械公司和食品加工企业。

## 2. 核心优势
1. **双模式（铝铝/铝塑）**：仅需更换成型模具即可切换工艺。铝铝采用冷冲压成型；铝塑采用PVC正压吹泡成型。一机两用，降低设备投资成本。

2. **快速定位模具系统**：区别于传统全拆卸模具，气动快夹设计将换模时间缩短30分钟以上，高度适应频繁换产需求。模具经超声波抛光处理，成型光滑无毛刺。

3. **多类型高精度加料系统**：可选配圆盘刷式、抽屉式、振动管道式和泡罩匹配式加料器。填充精度高达99.9%，适用于胶囊、异形片、大丸剂等多种剂型。

4. **三级在线缺粒检测+自动剔除**：可选机械/光纤/视觉检测。视觉型号可识别缺粒、破损和异物。信号联动卸料气缸自动剔除不合格板，确保高合格率。

5. **精密电子控制时序系统**：主轴编码器脉冲控制成型和吹气时序，取代传统凸轮。所有参数均可通过触摸屏调节。步进电机牵引，行程自由设定，色标对版精度±0.01mm，无膜片跑偏。

6. **恒温加热结构**：300mm PVC预热箱确保充分软化；停机时上下预热板自动分离。热封模具停机时自动抬起，避免高温灼烧包装材料，温控误差±1℃。

7. **模块化可调工位**：成型、封合、压痕和冲裁工位均配备手轮微调，带刻度定位和锁紧，调试方便。可选配撕裂线压痕工位和批号打印。

8. **稳定工业级配置**：品牌电气元件，包括三菱PLC、台达变频器、欧姆龙继电器、明纬电源。主电机带自锁刹车，基材断裂自动停机报警，配备完善的安全联锁和急停按钮。

9. **分体式设计，便于安装**：前后分体结构，便于在狭窄车间运输和安装。模具循环水冷却，防止高速连续生产时变形。

10. **自动废料收卷**：同步张紧泡罩基材，确保冲裁顺畅。边料成卷收集，保持车间清洁，减少人工清理。

## 4. 操作流程
1. **膜片安装与调整**
安装PVC/冷成型铝基材卷和PTP铝箔卷。调整放卷张力和导辊。铝塑模式开启预热，铝铝模式关闭预热。

2. **触摸屏参数设置**
设置成型温度、封合温度、速度、行程、编码器脉冲、剔除参数和批号打码信息。支持中英文界面切换。

3. **预热与空载试运行**
启动加热至设定温度，低速运行机器。检查加料、成型、牵引、封合、冲裁、废料收卷。测试色标对版、检测、急停和断料报警功能。

4. **物料装载与加料调整**
将片剂/胶囊加入料斗。调整振动频率和刷速，确保填充饱满。铝铝模式使用抽屉式/振动式加料器，铝塑模式使用圆盘刷式加料器。

5. **全自动生产**
启动自动运行。机器完成成型→填充→检测→封合/打码→（可选撕裂线）→对版→冲裁。缺陷品自动剔除；合格品收集；废料同步收卷。

6. **停机与清洁**
清空料斗，低速运行清除残留物料。关闭加热、气源、电机、电源。分离预热板和封合板。清洁模具、加料器和废料系统，进行维护保养。

### 常见问题
Q1：是否需要空压机？
A：需要。成型吹气、封合加压、冲裁脱模和剔除气缸均需压缩空气。稳定工作压力：0.6–0.8MPa。

Q2：铝铝和铝塑模式切换困难吗？
A：仅需更换成型模具。牵引、封合、冲裁工位不变。快夹设计允许一人10分钟内完成换模。

Q3：如何解决缺粒/漏装问题？
A：升级视觉检测；调整加料器振动/刷速并保持料斗充足；特殊胶囊/丸剂使用专用泡罩加料器。

Q4：如何调整纵向膜片跑偏？
A：在触摸屏上微调色标传感器牵引补偿。对版精度±0.01mm大幅减少人工调整。

Q5：模具可以单独定制吗？
A：可以。根据泡罩尺寸和数量定制模具。相同板面尺寸可共用冲裁模具以降低成本。

Q6：机器能否进入狭窄车间？
A：可以。分体式设计允许分体运输，现场组装。
