## 4. Operation Process
1. **Placement:** Arrange products neatly on the skin pack cardboard and place the board onto the worktable.
2. **Heating:** Pull down the film frame; the lamp heating system softens the film.
3. **Vacuum Sealing:** Start the vacuum pump; the softened film is drawn tight around the product and bonded to the cardboard by suction.
4. **Unloading:** Once cooled, open the tray frame and remove the finished package.

**Q2: Does this machine require an air compressor?**
A2: No. The SP-390 features a built-in powerful vacuum pump. It only needs a power connection to operate.

**Q3: Which voltage should I choose?**

**Q4: Is the packing speed fast?**
