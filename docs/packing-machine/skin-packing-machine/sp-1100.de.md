## 4. Betriebsablauf
1. **Vorbereitung:** Ordnen Sie die Produkte sauber auf der Skin-Pappe an und legen Sie diese auf den 1100 mm überdimensionierten Arbeitstisch.
2. **Folienaufweichung:** Senken Sie den Folienrahmen; das Lampenheizsystem erwärmt die Folie gleichmäßig, bis sie geschmeidig ist.
3. **Vakuumversiegelung:** Aktivieren Sie die Vakuumpumpe; die Luft wird schnell abgesaugt, wodurch die Folie auf der Pappe und dem Produkt haftet.
4. **Entnahme:** Lassen Sie die Folie nach dem Abkühlen abkühlen, öffnen Sie den Rahmen und entnehmen Sie die fertige Pappe zum Beschneiden.

### FAQ
**F1: Warum benötigt diese Maschine 8 KW Leistung?**
A1: Aufgrund der massiven Heizfläche (über 0,6 m²) ist eine hohe Leistung erforderlich, um sicherzustellen, dass die Lampen die gesamte Oberfläche schnell und gleichmäßig für eine gleichbleibende Qualität erwärmen.

**F2: Ist die Lampenheizung langsamer als die Drahtheizung?**
A2: Obwohl die Vorheizzeit etwas länger ist, bietet die Lampenheizung die "Gleichmäßigkeit", die für großflächige Verpackungen unerlässlich ist und ein Durchbrennen der Folie auf einer so großen Fläche verhindert.

**F3: Kann das Vakuum einen so langen Tisch bewältigen?**
A3: Absolut. Wir verwenden eine leistungsstarke industrielle Vakuumpumpe, die innerhalb von Sekunden den erforderlichen Unterdruck auf der gesamten Oberfläche erreicht.

**F4: Ist die Maschine leicht zu bewegen?**
A4: Trotz eines Gewichts von 416 kg ist sie mit schweren Industrierollen ausgestattet, die eine einfache Bewegung auf ebenen Werkstattböden ermöglichen.
