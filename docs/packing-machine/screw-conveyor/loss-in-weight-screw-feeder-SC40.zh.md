## 一、产品概述

* **类别**：工业自动化配料与给料系统 / 粉体定量给料机。
* **应用领域**：广泛应用于需要高精度连续或批次定量给料的行业，如化工、制药、食品、塑料改性（如母粒添加）及锂电池材料。特别适用于流动性差或易架桥的细粉或颗粒状物料。
* **核心功能**：利用动态称重技术，系统实时监测料斗的重量损失速率（失重逻辑）。PLC系统精确控制螺旋速度，实现精准、恒定的流量给料。

## 二、核心优势

1. **高精度配料控制**：配备工业级高灵敏度称重传感器，实时反馈重量变化。配料精度可达±1% - 2%，确保混合比例严格一致。
2. **304不锈钢结构**：与物料接触部件采用食品/制药级S304不锈钢制造。耐腐蚀、易清洁，符合国际卫生标准，防止物料污染。
3. **主动搅拌与防架桥技术**：配备集成式内部搅拌或防架桥机构，强力消除料斗内的“架桥”或“鼠洞”现象。确保粘性或超细粉末也能顺畅给料。
4. **智能PLC闭环系统**：系统根据目标给料速率自动调节变频电机转速，无需人工干预。支持与生产线其他设备联动。
5. **模块化定制**：出料口高度和连接接口可定制，以匹配您现有反应釜、包装机或混合机的高度。

## 三、操作流程

1. **物料装载**：将物料加入40L储料斗。
2. **参数设置**：在控制面板或PLC触摸屏上设置目标给料速度（如kg/h）或单批次配料重量。
3. **启动运行**：系统启动后，PLC读取传感器的初始重量。
4. **失重监测**：随着螺旋旋转给料，料斗内重量减少。PLC计算单位时间内的重量差。
5. **动态调节**：若实际失重率低于设定值，PLC自动提高螺旋速度；反之则降低，以保持恒定输出。
6. **自动停止/补料**：达到预设重量或触发低料位报警时，系统自动停止或提示补料。
