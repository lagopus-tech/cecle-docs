# 液体上料机 (TDSL)

---

## 一、产品概述

![液体上料机演示](https://cdn.shopify.com/s/files/1/0010/4982/1242/files/17825458761142.png?v=1782545914)

* **类别**：流体输送设备 / 气动泵 / 灌装辅助供料机
* **应用领域**：广泛应用于化妆品、个人护理、食品加工、制药及精细化工行业。主要用于从存储桶（如200L标准桶）中自动抽取液体原料，并将其输送至灌装机料斗、搅拌罐或其他高位容器。
* **核心功能**：采用全气动驱动原理，实现对低粘度及高粘度液体物料的自动化、防爆、高位垂直提升及稳定定量输送。

---

## 二、核心优势

* **全气动驱动，100%工业防爆**：完全无需用电，输送易燃易爆液体（如酒精、香精、溶剂）时具有极高的安全性。
* **自动抽取**：自动从深桶中抽取液体，大幅降低劳动强度，提升整线生产效率。
* **卓越的粘度适应性**：专为高粘度液体及膏体设计，可轻松抽取洗发水、蜂蜜、面霜等浓稠物料。
* **高级材质**：接触液体部件及外壳均采用304/316不锈钢制造，耐腐蚀、易清洗，符合卫生标准。
* **精准调节**：机身侧面配备调节手轮，可直观、实时地控制输送压力或液体流量。

---

## 三、操作流程

1. **设备定位**：将抽料管直接插入存储桶，并将机器移至目标容器旁。
2. **气源连接**：连接至符合压力要求（0.4–0.8 MPa）的工厂压缩空气管路。
3. **流量调节**：使用调节手轮预设压力及流量。
4. **自动输送**：打开气动开关，液体将通过不锈钢管及软管稳定注入目标料斗。

* **问：它能处理多高粘度的物料？**
  * 答：专为高粘度物料设计；可高效抽取从流动的洗发水到蜂蜜、面霜等浓稠膏体。
