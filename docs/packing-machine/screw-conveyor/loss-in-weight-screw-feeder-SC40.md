## I. Product Overview

* **Category**: Industrial Automated Dosing and Feeding Systems / Powder Quantitative Feeder.
* **Applications**: Widely used in industries requiring high-precision continuous or batch quantitative feeding, such as chemical, pharmaceutical, food, plastic compounding (e.g., masterbatch addition), and lithium battery materials. It is particularly suitable for fine powders or granular materials with poor flowability or those prone to bridging.
* **Core Function**: Utilizing dynamic weighing technology, the system monitors the real-time weight loss rate of the hopper (loss-in-weight logic). The PLC system precisely controls the screw speed to achieve accurate, constant flow feeding.

## II. Core Advantages

1. **High-Precision Dosing Control**: Equipped with industrial-grade high-sensitivity load cells for real-time feedback on weight changes. Dosing accuracy can reach ±1% - 2%, ensuring strict consistency in mixture ratios.
2. **304 Stainless Steel Construction**: Parts in contact with materials are made of food/pharmaceutical grade S304 stainless steel. It is corrosion-resistant, easy to clean, and complies with international hygiene standards to prevent material contamination.
3. **Active Agitation and Anti-Bridging Technology**: Features an integrated internal agitation or anti-bridging mechanism to forcefully eliminate "bridging" or "rat-holing" phenomena within the hopper. This ensures smooth feeding even for sticky or ultra-fine powders.
4. **Intelligent PLC Closed-Loop System**: The system automatically adjusts the variable frequency motor speed based on the target feeding rate, requiring no manual intervention. It supports linkage with other equipment in the production line.
5. **Modular Customization**: The discharge outlet height and connection interface can be customized to match the height of your existing reactors, packaging machines, or mixers.

## III. Operation Process

1. **Material Loading**: Add material into the 40L storage hopper.
2. **Parameter Setting**: Set the target feeding speed (e.g., kg/h) or single-batch dosing weight on the control panel or PLC touch screen.
3. **Start Operation**: Once the system starts, the PLC reads the initial weight from the sensors.
4. **Loss-in-Weight Monitoring**: As the screw rotates to feed, the weight in the hopper decreases. The PLC calculates the weight difference per unit of time.
5. **Dynamic Adjustment**: If the actual weight loss rate is lower than the set value, the PLC automatically increases the screw speed; otherwise, it decreases it to maintain a constant output.
6. **Automatic Stop/Refill**: Upon reaching the preset weight or triggering a low-level alarm, the system stops automatically or prompts for a refill.
