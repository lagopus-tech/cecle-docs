# 斗式提升机

---

## 一、产品概述

![斗式提升机](https://cdn.shopify.com/s/files/1/0010/4982/1242/files/buket.png?v=1782544848)

* **类别**：工业垂直输送设备/提升给料机。
* **应用领域**：广泛应用于食品、化工、农业及五金行业，用于将颗粒状、块状或小片状物料（如糖果、坚果、薯片、宠物食品）从低处垂直提升至包装机、料斗或称重设备的顶部。
* **核心功能**：利用链条或皮带驱动料斗，将物料从底部料斗提升至顶部并自动卸料，实现物料高度的自动化转移。

## 二、核心优势

1. **易清洁设计**：料斗采用可拆卸设计，极大方便日常清洗和消毒，尤其适用于卫生要求高的食品行业。
2. **高度灵活定制**：提供标准提升高度，并支持根据实际生产线布局定制非标高度。
3. **变频调速控制**：提升速度可通过变频器在0–15m/min范围内调节，以适应不同生产节奏。
4. **坚固耐用**：核心部件（包括PLC、电机、轴承、减速箱）均选用高规格组件，确保整机运行平稳。

## 三、操作流程

1. **进料**：将物料倒入底部料斗，或由上游设备连续供料。
2. **启动**：通过控制柜启动机器，变频器驱动电机运行。
3. **垂直提升**：在驱动机构带动下，料斗稳定地将物料向上输送。
4. **自动卸料**：到达顶部转向点时，料斗利用重力将物料精确卸入下游设备进料口。
5. **速度调节**：根据进料需求随时调节控制柜上的变频值，改变输送速度。
