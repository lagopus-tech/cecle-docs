## 三、操作流程
1. **部署与弯曲**：将吸粮机推至作业点，铺设柔性管，根据现场环境自由弯曲。
2. **通电测试**：接通电源，确认旋转方向。
3. **多场景吸料**：将吸料口插入粮堆或料仓内，开始吸料。
4. **快速清理与停机**：作业结束后，让设备空转1-2分钟排空余料，然后断开电源，收回软管。
