# 选型指南与产品概览：玻璃纸裹包机

## 一、什么是玻璃纸裹包机？

玻璃纸裹包机，又称外包膜机或“烟盒裹包机”，使用BOPP薄膜（可选配易撕线）为刚性长方体物品提供紧密的三维裹包。

*   **核心价值：** 防潮、防伪、提升美观度、节约成本（无需热收缩通道，避免损坏热敏性内容物）。
*   **目标行业：** 广泛应用于香烟、茶叶盒、化妆品、香水、药品、保健品、文具及小礼品盒包装。

### 核心工作流程
与传统的全自动机器相比，我们的设备采用**模块化创新**，将机器分为两个独立部分：**裹包单元**（前端）和**折叠单元**（后端），以实现更大的灵活性。

#### 1. 裹包单元（前端工序）
处理薄膜裹包、切割和初始封口：手动将BOPP薄膜360°包裹在产品上。设备自动切割薄膜并封合切割点，完成半成品裹包。
*   **代表型号：** `ACW-88`， `ACW-88A`

#### 2. 折叠单元（后端工序）
处理侧面折叠和封口：将已360°裹包的半成品盒子放入此单元，设备自动折叠、折入并封合剩余的两个开口端，完成三维裹包。
*   **代表型号：** `3DP-88`， `A3DP-88`

### 产品命名规则
1.  **基于ACW-88裹包单元：**
    *   **T** = 易撕线装置
    *   **F** = 薄膜废料收集装置
    *   **D** = 点烫装置
2.  **基于3DP-88折叠单元：**
    *   **A** = 自动（气动驱动）
    *   **H** = 加热板（上/下加热）
    *   **D** = 点烫装置

## 三、常见问题解答

### 1. 耗材
*   **问1：可以使用普通的PE/POF薄膜吗？**
    *   **答：** 不可以。必须使用**BOPP热封膜**。
*   **问2：推荐薄膜厚度是多少？**
    *   **答：** 21μm - 25μm 为最佳。
*   **问3：易撕线如何粘附？**
    *   **答：** 易撕线带有热激活胶；机器的张力器将其送至薄膜上，在热封过程中粘合。

### 2. 调整与质量
*   **问4：为什么会出现皱纹？**
    *   **答：** 通常是由于温度过低、速度过快或薄膜张力不均匀导致。
*   **问5：“点烫”和“六面封合”有什么区别？**
    *   **答：** 点烫在折叠过程中固定形状（防止滑移），而六面封合则在顶部和底部提供最终的“镜面效果”。

### 3. 硬件与选型
*   **问6：气动型号需要空压机吗？**
    *   **答：** 是的，一台小型静音无油空压机（0.6-0.8MPa）即可满足要求。
*   **问7：哪些型号需要模具？**
    *   **答：** ACW系列和A3DP-88A无需模具。3DP-88和工业型号通常需要定制模具。
