# Especificação do Produto

---

### I. Visão Geral do Produto
*   **Categoria:** Máquina Manual de Embrulho com Celofane / Embaladora de Caixas de Cigarro.
*   **Aplicações:** Amplamente utilizada para embalar com celofane (BOPP) caixas pequenas como perfumes, cosméticos, produtos farmacêuticos, chá, baralhos e caixas de cigarro. É ideal para produção em pequenos lotes, prototipagem, oficinas caseiras ou startups.
*   **Funções Principais:** Operação manual usando uma placa de aquecimento de liga de alumínio para selar o filme transparente na caixa do produto, obtendo um acabamento sofisticado e liso "estilo maço de cigarro" que proporciona proteção contra umidade, antifalsificação e aprimoramento estético.

5.  **Ampla Versatilidade:** Não requer troca de molde para diferentes tamanhos; uma única máquina pode lidar manualmente com várias especificações de caixas pequenas.

### VII. Fluxo de Trabalho Padrão
1.  **Pré-Aquecimento:** Ligue a máquina e ajuste a temperatura da placa de aquecimento, aguardando o sistema atingir o valor predefinido.
2.  **Preparação do Filme:** Prepare o filme transparente pré-cortado com base nas dimensões do produto.
3.  **Embrulho Manual:** Coloque o produto no centro do filme e faça a dobra manual inicial.
4.  **Selagem Lateral:** Pressione firmemente os lados do produto embalado contra a placa de aquecimento de alumínio para selar o filme instantaneamente.
5.  **Dobra de Cantos:** Imite o estilo de dobra de maço de cigarro para ambas as extremidades, pressionando contra a placa de aquecimento para moldar.
6.  **Finalização:** Verifique a planura da selagem e retire o produto perfeitamente embalado.
