# 产品规格书
**产品名称：** 玻璃纸包装机  
**型号：** CW-115

---

### 一、产品概述
*   **类别：** 手动玻璃纸包装机 / 烟盒外包机。
*   **应用：** 广泛应用于香水、化妆品、药品、茶叶、扑克牌及烟盒等小盒的玻璃纸（BOPP）包装。非常适合小批量生产、打样、家庭作坊或初创企业。
*   **核心功能：** 手动操作，使用铝合金加热板将透明薄膜热封在产品盒上，实现高端、光滑的“烟盒式”包装效果，提供防潮、防伪及美观提升。

---

### 二、核心优势
1.  **维护简单：** 全手动设计，无复杂电路或气动元件；结构坚固，故障率极低。
2.  **加热均匀：** 配备高品质**铝合金加热板**，导热快速均匀，确保封口平整、整洁，不损坏纸盒。
3.  **紧凑便携：** 仅重6公斤；占地面积小，便于运输和灵活操作。
4.  **高性价比：** 以极低的初始投资提升产品价值；单件包装成本超低。
5.  **通用性强：** 不同尺寸无需更换模具；一台机器可手动处理多种小盒规格。

### 七、标准工作流程
1.  **预热：** 接通电源并设置加热板温度，等待系统达到预设值。
2.  **备膜：** 根据产品尺寸准备预切透明薄膜。
3.  **手工包裹：** 将产品置于薄膜中心，进行初步手工折叠。
4.  **侧面封口：** 将包裹好的产品侧面紧压铝合金加热板，瞬间封合薄膜。
5.  **折角：** 模仿烟盒折叠方式处理两端，并压向加热板定型。
6.  **完成：** 检查封口平整度，取出包装完美的产品。
