### I. Обзор продукта
* **Категория продукта**: Автоматизированное упаковочное оборудование для пищевых/промышленных гранул
* **Сценарии использования**: Широко применяется для автоматической упаковки сыпучих материалов, таких как орехи, сухофрукты, конфеты, печенье, зерновые, корма для домашних животных, воздушные продукты, пластиковые гранулы и мелкие метизы. Подходит для предприятий глубокой переработки пищевых продуктов, первичной сельскохозяйственной переработки и промышленных упаковочных линий.
* **Основные функции**: Обеспечивает полностью автоматическую подачу пакетов, нанесение кода, открывание пакетов, высокоточное комбинационное взвешивание/дозирование, автоматическую термосварку и выдачу готовой продукции. Точный контроль веса наполнения обеспечивается интеллектуальной системой управления на базе ПЛК для гарантии стабильности упаковки.

### II. Ключевые преимущества
1. **Многомодельная адаптивность**: Доступны модели PBP-G200/250/300 для соответствия производственным требованиям для различных размеров гранул и пакетов.
2. **Высокоточное дозирование**: Оснащена высокочувствительным комбинационным весовым или объемным дозатором для эффективного контроля погрешности веса наполнения и снижения пересыпания или недосыпа материала.
3. **Конструкция против отходов**: Оснащена интеллектуальной системой обнаружения, которая автоматически останавливает наполнение и включает сигнализацию при отсутствии пакета, невозможности его открытия или дефекте сварного шва, предотвращая потери упаковочных материалов и продукта.
4. **Прочная конструкция**: Основной корпус изготовлен из высококачественной нержавеющей стали SUS304, которая устойчива к коррозии, долговечна и компактна, эффективно адаптируясь к высокочастотным, длительным производственным циклам.
5. **Быстрая переналадка**: Оснащена механизмом быстрой регулировки, позволяющим оперативно изменять ширину зажимов, обеспечивая быстрый переход между различными спецификациями продукции на одной производственной линии.

### VI. Технологический процесс
1. **Автоматическая подача пакетов**: Автоматический податчик пакетов перемещает готовый пакет к зажимам поворотного стола.
2. **Нанесение кода**: Автоматически выполняет печать даты производства и кода партии на поверхности пакета.
3. **Обнаружение открытия пакета**: Горловина пакета открывается с помощью вакуумных присосок; датчики выполняют обнаружение в реальном времени перед переходом к следующему этапу.
4. **Наполнение гранулами**: Дозирующее устройство точно подает отмеренное количество гранул в пакет.
5. **Термосварка**: Двухслойное нагревательное устройство выполняет термосварку горловины пакета, обеспечивая герметичный и ровный шов.
6. **Охлаждение и выгрузка**: Зона сварки охлаждается и фиксируется, готовое изделие плавно выгружается с помощью конвейерной ленты, завершая цикл.
