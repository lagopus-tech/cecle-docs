# 选型指南与产品概览：萨莫萨三角饺机

## 一、什么是萨莫萨三角饺机？

萨莫萨三角饺机是一种专门用于生产三角形萨莫萨（又称咖喱角或印度三角饺）的食品加工设备。它通过精确的机械运动模拟手工折叠技术，确保面皮与馅料完美密封，解决了手工生产中速度慢、封口不严、形状不一致等常见问题。

*   **核心价值：** 显著提升产能、降低人工成本、确保产品标准化，满足大规模冷冻食品供应或连锁餐厅的需求。
*   **目标行业：** 冷冻食品加工厂、中央厨房、快餐连锁店以及东南亚/印度特色餐厅。

---

## 二、产品分类与核心差异

我们的产品线按自动化程度分为：**半自动系列**（专注于折叠与封口）和**全自动系列**（从浆料到成品的完整端到端生产线）。

### 1. 半自动萨莫萨三角饺机（手动放皮）
该系列的核心是“机械替代手工折叠”，适用于拥有成熟面皮配方或使用市售春卷皮的客户。

*   **核心操作步骤：**
    1.  **手动裁切：** 用户准备矩形面皮（如春卷皮），并根据机器模具宽度进行裁切。
    2.  **手动放置：** 操作员将每张面皮精确放入进料口或模具槽中。
    3.  **自动注馅：** 机器感应到面皮后，自动泵入预设重量的馅料。
    4.  **自动折叠：** 机器通过机械臂完成“左折、右折、按压”动作，输出成品萨莫萨。
*   **主要特点：** 灵活性高，兼容多种手工面皮材料，初始投资较低，适合中小规模生产。

### 2. 全自动萨莫萨三角饺生产线（浆料进料）
这是一套真正的“无人化”生产方案，实现了从液态原料到成型萨莫萨的闭环生产。

*   **核心操作步骤：**
    1.  **原料准备：** 用户只需准备好混合好的**液态面浆**和**调制好的馅料**。
    2.  **自动制皮：** 机器利用加热旋转滚筒，将面浆瞬间凝固成连续的薄面皮带。
    3.  **自动裁切：** 内部机构自动将连续面皮带裁切成指定长度的条状。
    4.  **自动注馅：** 馅料被精确地分配到移动的面皮条上。
    5.  **全自动折叠：** 机器在流水线上高速完成多次连续折叠，将成品直接输送到速冻机或包装台。
*   **主要特点：** **无需任何手动裁切或放皮。** 只需监控浆料和馅料液位。产能巨大，卫生标准高，产品一致性极佳。

## 五、技术常见问题解答

### 1. 面皮与原料
*   **问1：半自动机器可以使用冷冻春卷皮吗？**
    *   **答：** 可以。只要解冻后保持韧性和粘性，半自动型号即可完美折叠。
*   **问2：全自动型号可以调节面皮厚度吗？**
    *   **答：** 可以。通过调节加热滚筒的间隙和速度，可将厚度控制在0.3mm至2mm之间。

### 2. 馅料与形状
*   **问3：馅料中可以含有大块土豆或肉块吗？**
    *   **答：** 由于泵送机构的原因，我们建议将颗粒控制在10mm以内，以防止堵塞注馅嘴。
*   **问4：成品的形状是标准三角形吗？**
    *   **答：** 是的。我们的模具模拟交替的90度和45度折叠，形成棱角分明、封口严密的三角形，油炸时不易爆裂。

### 3. 硬件与选型
*   **问5：哪种型号更容易清洁？**
    *   **答：** 全自动系列采用不锈钢和食品级输送带制造，配备快拆设计，符合工业卫生标准。
*   **问6：生产不同尺寸的萨莫萨需要多台机器吗？**
    *   **答：**
        *   **半自动：** 一台机器支持一种尺寸范围。
